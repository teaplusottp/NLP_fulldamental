## 1. Document Embeddings là gì?

- Biểu diễn **toàn bộ văn bản / tài liệu** thành **vector số thực**
- Mục tiêu:
  - Giữ semantic meaning của **cả đoạn văn hoặc tài liệu**
  - Dùng cho **semantic search, clustering, classification**
- Khác với sentence embeddings: capture ngữ nghĩa ở **nhiều câu, nhiều đoạn**

---

## 2. Ứng dụng

- Document Classification (topic / spam / sentiment)
- Semantic Search (tìm tài liệu giống câu hỏi)
- Document Clustering
- Recommendation system
- Question Answering

---

## 3. Các phương pháp

### 🔹 3.1 Averaging Sentence Embeddings
- Trung bình tất cả sentence embeddings trong document:

vec(document) = mean(vec(sentence1), vec(sentence2), ...)

- Ưu: đơn giản  
- Nhược: mất thứ tự, dài quá → mất thông tin

### 🔹 3.2 Doc2Vec (Paragraph Vector)
- Extension của Word2Vec
- Mỗi document có **vector riêng**
- Capture semantic và context document

### 🔹 3.3 Transformer / BERT Pooling
- Split document thành chunks, encode bằng **BERT / PhoBERT**
- Pooling (mean / max / CLS token) → document vector
- Hiệu quả với **semantic search & QA**

## 4. Code Python – Doc2Vec (gensim)

```python
from gensim.models.doc2vec import Doc2Vec, TaggedDocument

documents = [
    TaggedDocument(words=["tôi", "thích", "học", "AI"], tags=["doc1"]),
    TaggedDocument(words=["AI", "giúp", "tôi", "học", "nhanh"], tags=["doc2"])
]

model = Doc2Vec(documents, vector_size=50, window=2, min_count=1, epochs=20)

# Vector document
vec_doc1 = model.dv['doc1']
print("Vector doc1:", vec_doc1)

# Tìm document gần nhất với doc1
similar_docs = model.dv.most_similar('doc1')
print(similar_docs)
5. Code Python – Transformer-based
from sentence_transformers import SentenceTransformer, util

model = SentenceTransformer('all-MiniLM-L6-v2')

documents = [
    "Tôi thích học AI.",
    "AI giúp tôi học nhanh."
]

# Encode document
doc_embeddings = model.encode(documents, convert_to_tensor=True)

# Cosine similarity giữa 2 document
similarity = util.cos_sim(doc_embeddings[0], doc_embeddings[1])
print("Similarity:", similarity.item())
