# 📘 Chương 1 - Bài 5: Stemming & Lemmatization

---

## 1. Stemming & Lemmatization là gì?

Đây là 2 kỹ thuật giúp đưa từ về **dạng gốc (root form)**.

👉 Mục tiêu:
- Giảm số lượng từ
- Giúp model hiểu tốt hơn

---

## 2. Ví dụ

```
playing, played, plays
```

→ đưa về:

- Stemming → "play"
- Lemmatization → "play"

---

## 3. Stemming là gì?

**Stemming** là cắt từ theo rule đơn giản.

### Ví dụ:

```
studies → studi
playing → play
```

👉 Nhanh, đơn giản  
❌ nhưng có thể sai ngữ nghĩa

---

## 4. Lemmatization là gì?

**Lemmatization** đưa từ về dạng chuẩn theo từ điển.

### Ví dụ:

```
better → good
running → run
```

👉 Chính xác hơn  
❌ chậm hơn (cần dictionary)

---

## 5. So sánh

| | Stemming | Lemmatization |
|--|----------|---------------|
| Tốc độ | Nhanh | Chậm |
| Độ chính xác | Thấp | Cao |
| Dùng từ điển | Không | Có |

---

## 6. Code Python

---

### 🔹 Stemming (NLTK)

```python
from nltk.stem import PorterStemmer

stemmer = PorterStemmer()

words = ["playing", "studies", "running"]

for w in words:
    print(stemmer.stem(w))
```

---

### 🔹 Lemmatization (NLTK)

```python
from nltk.stem import WordNetLemmatizer

lemmatizer = WordNetLemmatizer()

words = ["playing", "better", "running"]

for w in words:
    print(lemmatizer.lemmatize(w))
```

---

## 7. Vấn đề thực tế

---

### ❌ Stemming sai nghĩa

```
"university" → "univers"
```

---

### ❌ Lemmatization cần POS tag

```
"running"
```

- noun → "running"
- verb → "run"

---

## 8. Tiếng Việt thì sao?

👉 Gần như KHÔNG dùng stemming

Vì:
- tiếng Việt không biến đổi từ nhiều như English

→ thường dùng:
- normalization
- tokenization

---

## 9. Khi nào dùng?

- Search engine → stemming (nhanh)
- NLP chuẩn → lemmatization
- Deep learning → thường không cần
