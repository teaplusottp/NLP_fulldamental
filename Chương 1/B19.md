# 📘 Chương 1 - Bài 19: Chunking & Shallow Parsing trong NLP

---

## 1. Chunking là gì?

**Chunking** (còn gọi **Shallow Parsing**) = gom các token thành **các cụm từ (phrase)** dựa trên POS Tagging:

- NP = Noun Phrase (Cụm danh từ)  
- VP = Verb Phrase (Cụm động từ)  
- PP = Prepositional Phrase (Cụm giới từ)

---

## 2. Ví dụ

Text:

"Tôi thích học AI tại Hà Nội"


### POS Tagging:
| Token | POS |
|-------|-----|
| Tôi   | PRON |
| thích | VERB |
| học   | VERB |
| AI    | NOUN |
| tại   | ADP |
| Hà    | PROPN |
| Nội   | PROPN |

---

### Chunking:
- NP: "Tôi", "AI", "Hà Nội"  
- VP: "thích học"  
- PP: "tại Hà Nội"

---

## 3. Tại sao Chunking quan trọng?

- Hiểu **cấu trúc câu** mà không cần full parse tree  
- Giúp **trích xuất cụm từ quan trọng**  
- Hỗ trợ:
  - NER
  - Relation extraction
  - Summarization

---

## 4. Phương pháp Chunking

### 🔹 Rule-based
- Dùng **POS patterns**
- Ví dụ NP: `(DET)? (ADJ)* NOUN+`

### 🔹 ML-based
- Học từ corpus annotated chunk
- Ví dụ: CRF, BiLSTM-CRF

### 🔹 Deep Learning
- Transformer-based: BERT, PhoBERT → fine-tune cho chunking

---

## 5. Code Python – NLTK (Rule-based)

```python
import nltk
from nltk import pos_tag, word_tokenize
from nltk.chunk import RegexpParser

text = "I love studying AI at Hanoi"
tokens = word_tokenize(text)
tags = pos_tag(tokens)

# Define chunk grammar
grammar = "NP: {<DT>?<JJ>*<NN.*>+}"
cp = RegexpParser(grammar)

tree = cp.parse(tags)
print(tree)
tree.draw()
Kết quả: highlight các Noun Phrase (NP)
VP, PP cũng tương tự nhưng thay grammar pattern
6. Chunking trong tiếng Việt
