## 1. Jaccard Similarity là gì?

**Jaccard Similarity** đo độ giống nhau giữa 2 tập hợp:

👉 dựa trên:
- phần giao (intersection)
- phần hợp (union)

---

## 2. Công thức


J(A, B) = |A ∩ B| / |A ∪ B|


Trong đó:
- A, B = tập từ (set)
- | | = số phần tử

---

## 3. Ví dụ đơn giản

### Câu:


A: "tôi thích AI"
B: "tôi thích học AI"


---

### 🔹 Convert thành set


A = {"tôi", "thích", "AI"}
B = {"tôi", "thích", "học", "AI"}


---

### 🔹 Tính toán

- Intersection:

{"tôi", "thích", "AI"} → 3


- Union:

{"tôi", "thích", "AI", "học"} → 4


---

👉 Jaccard:

3 / 4 = 0.75


---

## 4. Giá trị Jaccard

| Giá trị | Ý nghĩa |
|--------|--------|
| 1 | giống hoàn toàn |
| 0 | không liên quan |

---

## 5. Trực giác quan trọng

👉 Jaccard chỉ quan tâm:
- từ có xuất hiện hay không

👉 KHÔNG quan tâm:
- số lần xuất hiện
- thứ tự

---

## 6. Code Python

```python
def jaccard_similarity(s1, s2):
    set1 = set(s1.split())
    set2 = set(s2.split())
    
    intersection = set1 & set2
    union = set1 | set2
    
    return len(intersection) / len(union)


print(jaccard_similarity(
    "toi thich ai",
    "toi thich hoc ai"
))
```
