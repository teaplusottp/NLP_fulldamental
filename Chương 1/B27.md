---

## 1. Sentence Embeddings là gì?

- Biểu diễn **cả câu hoặc đoạn văn** thành **vector số thực**.
- Mục tiêu:
  - Giữ semantic meaning của cả câu
  - Cho phép **so sánh, clustering, retrieval**  
- Khác với word embeddings: sentence embeddings capture **ngữ cảnh toàn câu**.

---

## 2. Ứng dụng

- Semantic similarity: đo độ gần nghĩa giữa câu
- Sentence clustering
- Document classification
- Question Answering
- Information retrieval

---

## 3. Các phương pháp

### 🔹 3.1 Averaging Word Embeddings
- Trung bình các vector từ trong câu:

vec(sentence) = mean(vec(word1), vec(word2), ...)

- Ưu: đơn giản, nhẹ
- Nhược: mất thứ tự từ và context

### 🔹 3.2 Pretrained Sentence Embeddings
- **Universal Sentence Encoder (USE)**
- **Sentence-BERT (SBERT)**
- **PhoBERT / BERT pooling**
- Ưu: capture ngữ nghĩa tốt, giữ context
- Nhược: cần GPU để train/fine-tune

### 🔹 3.3 Transformer-based pooling
- Dùng BERT / PhoBERT, lấy vector **[CLS] token** hoặc mean pooling
- Capture **contextual semantics** tốt hơn

---

## 4. Code Python – Sentence-BERT (semantic similarity)

```python
from sentence_transformers import SentenceTransformer, util

# Load pretrained model
model = SentenceTransformer('all-MiniLM-L6-v2')

sentences = [
  "Tôi thích học AI.",
  "Học AI làm tôi vui."
]

# Encode sentences
embeddings = model.encode(sentences)

# Cosine similarity
similarity = util.cos_sim(embeddings[0], embeddings[1])
print("Similarity:", similarity.item())
Output: similarity score ~ [0,1], càng gần 1 → câu giống nghĩa
5. Code Python – Averaging Word Embeddings
import numpy as np
from gensim.models import Word2Vec

sentences_tokens = [
    ["tôi", "thích", "học", "AI"],
    ["AI", "giúp", "tôi", "học", "nhanh"]
]

model = Word2Vec(sentences_tokens, vector_size=50, window=2, min_count=1, sg=1)

def sentence_vector(sentence_tokens):
    vectors = [model.wv[w] for w in sentence_tokens if w in model.wv]
    return np.mean(vectors, axis=0)

vec1 = sentence_vector(sentences_tokens[0])
vec2 = sentence_vector(sentences_tokens[1])

sim = np.dot(vec1, vec2) / (np.linalg.norm(vec1)*np.linalg.norm(vec2))
print("Similarity:", sim)
