## 1. Cosine Similarity là gì?

**Cosine Similarity** là cách đo **độ giống nhau giữa 2 vector**.

👉 Trong NLP:
- mỗi câu → vector (BoW / TF-IDF)
- dùng cosine để so sánh

---

## 2. Ý tưởng trực quan

👉 So sánh **góc giữa 2 vector**

- Góc nhỏ → giống nhau
- Góc lớn → khác nhau

---

## 3. Công thức


cos(θ) = (A · B) / (||A|| × ||B||)


Trong đó:
- A · B = tích vô hướng
- ||A|| = độ dài vector A

---

## 4. Giá trị cosine

| Giá trị | Ý nghĩa |
|--------|--------|
| 1 | giống hoàn toàn |
| 0 | không liên quan |
| -1 | đối lập |

👉 NLP thường nằm trong:

0 → 1


---

## 5. Ví dụ đơn giản

### Vector:


A = [1, 1, 0]
B = [1, 1, 1]


👉 Cosine sẽ cao → vì khá giống nhau

---

## 6. Code Python

```python
from sklearn.metrics.pairwise import cosine_similarity

A = [[1, 1, 0]]
B = [[1, 1, 1]]

similarity = cosine_similarity(A, B)

print(similarity)
