## 1. Semantic Search là gì?

- Khác với **keyword search**, semantic search dựa vào **nghĩa của câu / văn bản**, không chỉ matching từ.
- Cho phép:
  - Tìm câu / document gần nghĩa với query
  - Bỏ qua exact words, vẫn tìm đúng kết quả

---

## 2. Information Retrieval (IR)

- IR = quá trình **tìm kiếm thông tin có liên quan** trong corpus
- Input: query
- Output: ranked list các document / câu
- Semantic search = IR + embeddings semantic

---

## 3. Pipeline Semantic Search

1. **Corpus preprocessing**: tokenize, clean, sentence split
2. **Encode documents**: Sentence / Document embeddings
3. **Encode query**: query → embedding
4. **Compute similarity**: cosine similarity giữa query và documents
5. **Rank & Return top-k results**

---

## 4. Cosine Similarity – Recap

\[
\text{similarity} = \frac{\vec{query} \cdot \vec{doc}}{||\vec{query}|| \times ||\vec{doc}||}
\]

- Score càng cao → semantic càng gần nhau
- Popular metric trong semantic search

---

## 5. Code Python – Semantic Search với SBERT

```python
from sentence_transformers import SentenceTransformer, util

# Load pretrained sentence embedding model
model = SentenceTransformer('all-MiniLM-L6-v2')

# Corpus documents
documents = [
    "Tôi thích học AI và Machine Learning.",
    "Học deep learning giúp phát triển AI.",
    "Tôi đi chơi cuối tuần."
]

# Encode corpus
doc_embeddings = model.encode(documents, convert_to_tensor=True)

# Query
query = "Làm sao học AI hiệu quả?"
query_embedding = model.encode(query, convert_to_tensor=True)

# Compute cosine similarity
cos_scores = util.cos_sim(query_embedding, doc_embeddings)

# Top result
top_result_idx = cos_scores.argmax()
print("Top document:", documents[top_result_idx])
