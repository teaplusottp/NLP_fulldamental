## 1. Edit Distance là gì?

**Edit Distance** là số bước ít nhất để biến một chuỗi thành chuỗi khác.

👉 Các phép biến đổi:
- Insert (thêm ký tự)
- Delete (xóa ký tự)
- Replace (thay ký tự)

---

## 2. Ví dụ trực quan

### Ví dụ 1:


"cat" → "cut"


👉 Thay:

a → u


→ Distance = 1

---

### Ví dụ 2:


"kitten" → "sitting"


👉 Các bước:
- kitten → sitten (k → s)
- sitten → sittin (e → i)
- sittin → sitting (add g)

→ Distance = 3

## 5. Cách tính (ý tưởng)

Dùng **Dynamic Programming**

👉 Tạo bảng (matrix)

Ví dụ:

"cat" vs "cut"


|   |  | c | u | t |
|---|--|---|---|---|
|   | 0| 1 | 2 | 3 |
| c | 1| 0 | 1 | 2 |
| a | 2| 1 | 1 | 2 |
| t | 3| 2 | 2 | 1 |

→ kết quả = ô cuối = **1**

## 6. Code Python

```python
def edit_distance(s1, s2):
    m, n = len(s1), len(s2)
    
    dp = [[0]*(n+1) for _ in range(m+1)]
    
    for i in range(m+1):
        dp[i][0] = i
    for j in range(n+1):
        dp[0][j] = j
        
    for i in range(1, m+1):
        for j in range(1, n+1):
            if s1[i-1] == s2[j-1]:
                cost = 0
            else:
                cost = 1
                
            dp[i][j] = min(
                dp[i-1][j] + 1,      # delete
                dp[i][j-1] + 1,      # insert
                dp[i-1][j-1] + cost  # replace
            )
    
    return dp[m][n]


print(edit_distance("kitten", "sitting"))
