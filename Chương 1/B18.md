# 📘 Chương 1 - Bài 18: Named Entity Recognition (NER) trong NLP

---

## 1. NER là gì?

**NER (Named Entity Recognition)** = xác định các **thực thể có tên** trong văn bản.

- Nhận biết:  
  - Person (Người)  
  - Organization (Tổ chức)  
  - Location (Địa điểm)  
  - Date / Time (Thời gian)  
  - Money, Percent, Product,…

---

## 2. Ví dụ

Text:

"Nguyễn Văn A làm việc tại VinGroup ở Hà Nội từ 2020"


### Kết quả NER:
- Nguyễn Văn A → PERSON  
- VinGroup → ORG  
- Hà Nội → LOCATION  
- 2020 → DATE

---

## 3. Tại sao NER quan trọng?

- Trích xuất thông tin từ văn bản
- Phục vụ:
  - Search / Indexing
  - Chatbot / QA
  - Summarization
  - Knowledge Graph

---

## 4. Phương pháp NER

### 🔹 4.1 Rule-based
- Dùng regex + từ điển
- Ưu điểm: nhanh, đơn giản
- Nhược điểm: khó mở rộng, dễ sai

### 🔹 4.2 ML-based
- Học từ labeled data
- Ví dụ: CRF, HMM
- Cần corpus annotated

### 🔹 4.3 Deep Learning
- BiLSTM + CRF
- Transformer (BERT-based) → state-of-the-art

---

## 5. Code Python (SpaCy – nhanh nhất)

```python
import spacy

nlp = spacy.load("en_core_web_sm")  # hoặc 'vi_core_news_sm' cho tiếng Việt

text = "Nguyễn Văn A làm việc tại VinGroup ở Hà Nội từ 2020"

doc = nlp(text)

for ent in doc.ents:
    print(ent.text, ent.label_)
Output:
Nguyễn Văn A PERSON
VinGroup ORG
Hà Nội GPE
2020 DATE
