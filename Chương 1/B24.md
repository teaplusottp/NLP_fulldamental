# 📘 Chương 2 - Bài 24: Word Embeddings trong NLP

---

## 1. Word Embeddings là gì?

**Word Embedding** = biểu diễn từ dưới dạng **vector số thực** trong không gian nhiều chiều, sao cho:

- Những từ **ngữ nghĩa gần nhau** có vector gần nhau
- Giúp máy tính hiểu **nghĩa từ / mối quan hệ từ**  

---

### Ví dụ:
- “king” - “man” + “woman” ≈ “queen”
- “apple” gần với “orange”, “banana” nhưng xa “car”

---

## 2. Lợi ích của Word Embeddings

- Giải quyết **sparse representation** của BoW / TF-IDF  
- Giúp **machine learning / deep learning models** học hiểu ngôn ngữ tốt hơn  
- Capture semantic + syntactic similarity  

---

## 3. Các loại Word Embeddings

### 🔹 3.1 Predictive Embeddings (Word2Vec)
- Học bằng **predict context** hoặc **predict word**  
- 2 kiểu:
  - CBOW (Continuous Bag-of-Words): predict word dựa trên context
  - Skip-gram: predict context dựa trên word

### 🔹 3.2 Count-based Embeddings (GloVe)
- Dựa trên **word co-occurrence matrix**  
- Học vector sao cho preserve **global statistics**

### 🔹 3.3 Contextual Embeddings (BERT, PhoBERT)
- Vector phụ thuộc **ngữ cảnh câu**
- “bank” trong “river bank” ≠ “bank” trong “money bank”

---

## 4. Code Python – Word2Vec (gensim)

```python
from gensim.models import Word2Vec

sentences = [
    ["tôi", "thích", "học", "AI"],
    ["AI", "giúp", "tôi", "học", "nhanh"]
]

model = Word2Vec(sentences, vector_size=50, window=2, min_count=1, sg=1)

# Vector của từ
vector_ai = model.wv['AI']
print(vector_ai)

# Từ gần nhất với "học"
print(model.wv.most_similar('học'))
5. Code Python – GloVe (pretrained)
import numpy as np

# Load pretrained GloVe vectors
glove_path = "glove.6B.50d.txt"
embeddings_index = {}

with open(glove_path, encoding="utf8") as f:
    for line in f:
        values = line.split()
        word = values[0]
        coefs = np.asarray(values[1:], dtype='float32')
        embeddings_index[word] = coefs

print("Vector for 'computer':", embeddings_index['computer'])

GloVe	Capture global statistics	Cố định vector, tốn memory
BERT/PhoBERT	Contextual, state-of-the-art	Nặng, cần GPU, phức tạp
