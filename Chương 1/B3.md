# 📘 Chương 1 - Bài 3: Text Normalization (Làm sạch văn bản)

---

## 1. Text Normalization là gì?

**Text Normalization** là quá trình làm sạch và chuẩn hóa văn bản trước khi đưa vào model.

👉 Mục tiêu:
- Giảm nhiễu (noise)
- Chuẩn hóa dữ liệu
- Giúp model học dễ hơn

---

## 2. Ví dụ trực quan

Input bẩn:
```
"Tôi   thích AI!!! vl 😆"
```

Sau khi normalize:
```
"toi thich ai"
```

---

## 3. Các bước phổ biến

### 🔹 3.1 Lowercase (viết thường)

```
"Hello AI" → "hello ai"
```

---

### 🔹 3.2 Remove punctuation

```
"Hello!!!" → "Hello"
```

---

### 🔹 3.3 Remove extra spaces

```
"Tôi   thích" → "Tôi thích"
```

---

### 🔹 3.4 Remove stopwords

```
"Tôi rất thích AI"
→ ["thích", "AI"]
```

---

### 🔹 3.5 Remove emoji

```
"Hay quá 😂" → "Hay quá"
```

---

### 🔹 3.6 Normalize slang

```
"ko", "vl", "ngon vãi"
→ "không", "rất", "rất ngon"
```

---

## 4. Khi nào KHÔNG nên normalize mạnh?

❗ Không phải lúc nào cũng xóa hết:

- Sentiment analysis → giữ "!!!", emoji
- Chatbot → giữ ngữ cảnh tự nhiên
- Gen AI → giữ style

---

## 5. Pipeline chuẩn

```
Text
→ Lowercase
→ Remove punctuation
→ Tokenize
→ Remove stopwords
→ Output sạch
```

---

## 6. Code Python demo

```python
import re

text = "Tôi   thích AI!!! vl 😆"

# lowercase
text = text.lower()

# remove punctuation
text = re.sub(r'[^\w\s]', '', text)

# remove extra spaces
text = re.sub(r'\s+', ' ', text).strip()

print(text)
```

Output:
```
"tôi thích ai vl"
```
