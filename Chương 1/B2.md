# 📘 Chương 1 - Bài 2: Tokenization (Tách từ)

---

## 1. Tokenization là gì?

**Tokenization** là bước chia văn bản thành các đơn vị nhỏ hơn gọi là **token**.

👉 Token có thể là:

* từ (word)
* ký tự (character)
* subword (một phần của từ)

---

### Ví dụ:

```text
"Tôi thích học AI"
```

→ Token:

```text
["Tôi", "thích", "học", "AI"]
```

---

## 2. Tại sao Tokenization quan trọng?

Model **không hiểu text raw**, nó chỉ hiểu số.

👉 Muốn xử lý:

```
Text → Token → Vector → Model
```

❗ Nếu tokenize sai → model hiểu sai luôn

---

## 3. Các loại Tokenization

---

### 🔹 3.1 Word Tokenization (theo từ)

```text
"I love NLP"
→ ["I", "love", "NLP"]
```

✔ đơn giản
❌ không xử lý tốt từ mới (OOV)

---

### 🔹 3.2 Character Tokenization (theo ký tự)

```text
"AI"
→ ["A", "I"]
```

✔ không bị OOV
❌ sequence rất dài, khó học

---

### 🔹 3.3 Subword Tokenization (quan trọng nhất hiện nay)

```text
"playing"
→ ["play", "##ing"]
```

✔ xử lý từ mới tốt
✔ dùng trong BERT, GPT

---

## 4. Tokenization trong tiếng Việt (CỰC KỲ QUAN TRỌNG)

Tiếng Anh:

```text
"I love AI"
```

Tiếng Việt:

```text
"Tôi thích học AI"
```

👉 nhìn giống nhau, nhưng:

```text
"sinh viên"
```

❗ thực chất là **1 từ**, nhưng có dấu cách

---

### Sai nếu tokenize kiểu space:

```text
["sinh", "viên"] ❌
```

### Đúng:

```text
["sinh_viên"] ✅
```

## 6. Demo code Python

---

### 🔹 English (NLTK)

```python
from nltk.tokenize import word_tokenize

text = "I love NLP"
tokens = word_tokenize(text)

print(tokens)
```

---

### 🔹 Vietnamese (underthesea)

```python
from underthesea import word_tokenize

text = "Tôi là sinh viên"
tokens = word_tokenize(text)

print(tokens)
```

👉 Output:

```text
["Tôi", "là", "sinh_viên"]
```

## 7. Một số vấn đề thực tế

---

```text

```

---

### ❌ 2. Emoji

```text

```

→ cần decide giữ hay bỏ

---

### ❌ 3. Slang

```text
"vl", "ko", "ngon vãi"
```

---

### ❌ 4. Mixed language

```text
"Code AI mệt vl"
```
