---

## 2. Cosine Similarity

- Đo độ tương đồng giữa 2 vector
- Công thức:

\[
\text{cosine\_similarity} = \frac{\vec{A} \cdot \vec{B}}{||\vec{A}|| \times ||\vec{B}||}
\]

- Giá trị: [-1, 1]  
  - 1 → vectors giống hệt  
  - 0 → vectors vuông góc, không liên quan  
  - -1 → vectors đối lập

---

## 3. Visualize Embeddings

- Dùng **PCA** hoặc **t-SNE** để giảm chiều vector về 2D/3D
- Plot ra scatter plot → xem clusters, từ gần nghĩa

---

## 4. Code Python – Cosine Similarity (Word2Vec)

```python
from gensim.models import Word2Vec
from sklearn.metrics.pairwise import cosine_similarity

sentences = [
    ["tôi", "thích", "học", "AI"],
    ["AI", "giúp", "tôi", "học", "nhanh"]
]

model = Word2Vec(sentences, vector_size=50, window=2, min_count=1, sg=1)

vec_ai = model.wv['AI']
vec_hoc = model.wv['học']

sim = cosine_similarity([vec_ai], [vec_hoc])
print("Similarity AI - học:", sim[0][0])
5. Code Python – Visualization (t-SNE)
import matplotlib.pyplot as plt
from sklearn.manifold import TSNE

words = ['tôi', 'thích', 'học', 'AI', 'nhanh']
vectors = [model.wv[w] for w in words]

tsne = TSNE(n_components=2, random_state=42)
vectors_2d = tsne.fit_transform(vectors)

plt.figure(figsize=(6,6))
for i, word in enumerate(words):
    x, y = vectors_2d[i]
    plt.scatter(x, y)
    plt.text(x+0.01, y+0.01, word)
plt.show()
