# 📘 Chương 1 - Bài 1: NLP là gì?

## 1. NLP là gì?

**NLP (Natural Language Processing)** là lĩnh vực trong AI giúp máy tính **hiểu, xử lý và tạo ra ngôn ngữ tự nhiên của con người**.

👉 Nói đơn giản:

* Người nói: "Hôm nay trời nóng quá"
* Máy phải hiểu: **đây là câu than phiền về thời tiết**

---

## 2. Tại sao NLP khó?

Ngôn ngữ con người rất "lộn xộn":

### ❌ Ambiguity (đa nghĩa)

* "bank"

  * ngân hàng
  * bờ sông

### ❌ Context (ngữ cảnh)

* "Tao thấy nó ngầu"

  * ngầu = cool 😎 (tích cực)
  * không phải "lạnh"

### ❌ Grammar không cố định

* "ăn cơm chưa?"
* "chưa ăn cơm à?"
  → cùng nghĩa, khác cấu trúc

---

## 3. NLP dùng để làm gì?

### 🔹 Ứng dụng phổ biến:

#### 1. Phân loại văn bản (Text Classification)

* Spam mail
* Phân loại tin tức

#### 2. Sentiment Analysis

* "Phim này hay vãi" → Positive
* "Chán vl" → Negative

#### 3. Chatbot

* ChatGPT 😏
* CSKH tự động

#### 4. Machine Translation

* Google Translate

#### 5. Tóm tắt văn bản

* Rút gọn bài báo

#### 6. Nhận diện thực thể (NER)

* "Hà Nội" → Location
* "Messi" → Person

---

## 4. Pipeline NLP cơ bản

Một hệ NLP thường đi theo pipeline:

```
Text → Preprocessing → Vectorization → Model → Output
```

### Ví dụ:

```
Input: "Tôi thích AI"

→ Tokenize: ["Tôi", "thích", "AI"]

→ Vector hóa: [0.2, 0.8, 0.5]

→ Model → Output: Positive
```

---

## 5. NLP vs Machine Learning vs Deep Learning

| Khái niệm | Ý nghĩa          |
| --------- | ---------------- |
| AI        | Trí tuệ nhân tạo |
| ML        | Máy học          |
| DL        | Deep Learning    |
| NLP       | Xử lý ngôn ngữ   |

👉 Quan hệ:

```
AI
 └── ML
      └── DL
           └── NLP (ứng dụng)
```

## 6. Các hướng tiếp cận NLP

### 🔹 Rule-based (cổ điển)

* Dùng luật
* Ví dụ: nếu có "không" → negative

❌ yếu khi data phức tạp

---

### 🔹 Machine Learning

* Học từ data
* Ví dụ: Naive Bayes, SVM

---

### 🔹 Deep Learning (hiện đại)

* Neural network
* Transformer, LLM

---

## 7. Ví dụ thực tế (quan trọng)

### Input:

```
"Phim này không hay lắm"
```

### Máy phải hiểu:

* Có từ "không"
* "không hay" → negative

👉 Đây là bài toán **Sentiment Analysis**
