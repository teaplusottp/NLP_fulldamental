# 📘 Chương 1 - Bài 6: N-gram (Cực kỳ quan trọng)

---

## 1. N-gram là gì?

**N-gram** là cách chia câu thành các cụm gồm **N token liên tiếp**.

👉 Hiểu đơn giản:
- N = 1 → từng từ (unigram)
- N = 2 → cặp từ (bigram)
- N = 3 → cụm 3 từ (trigram)

---

## 2. Ví dụ

Câu:

"I love NLP"


### 🔹 Unigram (N=1)

["I", "love", "NLP"]


### 🔹 Bigram (N=2)

["I love", "love NLP"]


### 🔹 Trigram (N=3)

["I love NLP"]


---

## 3. Tại sao N-gram quan trọng?

👉 Vì nó giữ được **ngữ cảnh (context)**

### Ví dụ:


"không thích"


- Unigram:

["không", "thích"]

→ dễ hiểu sai

- Bigram:

["không thích"]

→ hiểu đúng là **negative**

👉 Unigram thường không đủ để hiểu ý nghĩa câu

---

## 4. Công thức số lượng N-gram

Với câu có **T token**:


Số N-gram = T - N + 1


---

## 5. Ví dụ tính toán

Câu:

"Tôi rất thích học AI"


Có 5 từ:

- Unigram: 5
- Bigram: 5 - 2 + 1 = 4
- Trigram: 5 - 3 + 1 = 3

---

## 6. Code Python

### 🔹 Tạo N-gram với NLTK

```python
from nltk.util import ngrams

sentence = "I love NLP"
tokens = sentence.split()

bigrams = list(ngrams(tokens, 2))

print(bigrams)
```
👉 Output:

[('I', 'love'), ('love', 'NLP')]
