## 1. Text Classification là gì?

**Text Classification** = gán nhãn cho văn bản dựa trên nội dung.

👉 Ví dụ:
- Email → spam / not spam  
- Tin tức → tech / sports / politics  
- Review → positive / negative / neutral

## 6. Code Python (TF-IDF + Naive Bayes)

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import MultinomialNB

texts = [
    "Tôi thích sản phẩm này",
    "Giao hàng tệ quá",
    "Tin tức công nghệ hôm nay"
]

labels = ["positive", "negative", "tech"]

vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(texts)

X_train, X_test, y_train, y_test = train_test_split(X, labels, test_size=0.3)

model = MultinomialNB()
model.fit(X_train, y_train)

pred = model.predict(vectorizer.transform(["Sản phẩm rất tốt"]))
print(pred)
