## 3. Pipeline

1. **Text preprocessing**
   - Lowercasing, remove punctuation
   - Tokenization (ViTokenizer hoặc simple split)
2. **Word Embeddings**
   - Train Word2Vec / GloVe hoặc dùng pretrained
   - Optional: kiểm tra similarity / analogy
3. **Sentence Embeddings**
   - SBERT hoặc averaging word embeddings
4. **Document Embeddings**
   - Pool sentence embeddings hoặc Doc2Vec
5. **Semantic Search**
   - Encode query → compute cosine similarity → top-k documents
6. **Visualization**
   - t-SNE / PCA → plot documents
   - Clustering bằng KMeans / hierarchical clustering

---

## 4. Code Skeleton (Python)

```python
from gensim.models import Word2Vec, Doc2Vec, TaggedDocument
from sentence_transformers import SentenceTransformer, util
import matplotlib.pyplot as plt
from sklearn.manifold import TSNE
from sklearn.cluster import KMeans

# 1. Corpus
documents = [
    "Tôi thích học AI.",
    "Machine Learning giúp phát triển trí tuệ nhân tạo.",
    "Deep Learning là nhánh của AI.",
    "Cuối tuần tôi đi chơi với bạn bè."
]

# 2. Word Embeddings
sentences_tokens = [doc.lower().split() for doc in documents]
w2v_model = Word2Vec(sentences_tokens, vector_size=50, window=2, min_count=1, sg=1)

# 3. Sentence Embeddings
sbert_model = SentenceTransformer('all-MiniLM-L6-v2')
sentence_embeddings = sbert_model.encode(documents)

# 4. Document Embeddings (average pooling)
import numpy as np
doc_embeddings = np.array([np.mean([w2v_model.wv[w] for w in doc.lower().split() if w in w2v_model.wv], axis=0) for doc in documents])

# 5. Semantic Search
query = "Học trí tuệ nhân tạo"
query_emb = sbert_model.encode([query])[0]
cos_scores = util.cos_sim(query_emb, sentence_embeddings)
top_idx = cos_scores.argmax()
print("Top document:", documents[top_idx])

# 6. Visualization
tsne = TSNE(n_components=2, random_state=42)
vectors_2d = tsne.fit_transform(doc_embeddings)
plt.figure(figsize=(6,6))
for i, doc in enumerate(documents):
    x, y = vectors_2d[i]
    plt.scatter(x, y)
    plt.text(x+0.01, y+0.01, doc)
plt.show()

# 7. Optional: KMeans clustering
kmeans = KMeans(n_clusters=2, random_state=42).fit(doc_embeddings)
print("Cluster labels:", kmeans.labels_)
5. Mini bài tập thực hành
