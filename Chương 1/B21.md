# 📘 Chương 1 - Bài 21: Coreference Resolution trong NLP

---

## 1. Coreference Resolution là gì?

**Coreference Resolution** = xác định các từ/cụm từ **tham chiếu cùng một entity** trong văn bản.

- Ví dụ:  

"Nguyễn Văn A đi học. Anh ấy thích AI."

- "Nguyễn Văn A" ↔ "Anh ấy"  

→ Đây là **coreference pair**

---

## 2. Tại sao Coreference quan trọng?

- Giúp **nắm ý nghĩa liên tục trong văn bản**  
- Hỗ trợ:
  - Question Answering
  - Summarization
  - Chatbot / Dialogue system
  - Relation extraction

---

## 3. Ví dụ trực quan

Text:

"Minh và Lan đi siêu thị. Cô ấy mua sữa, anh ấy mua bánh."


### Coreference pairs:
- "Lan" ↔ "Cô ấy"
- "Minh" ↔ "Anh ấy"

---

## 4. Phương pháp Coreference Resolution

### 🔹 Rule-based
- Dùng pronoun + gender + number + nearby noun  
- Ví dụ: "anh ấy" → nam, số ít, gần "Minh" → link

### 🔹 ML-based
- Học từ corpus annotated  
- CRF, mention ranking, pairwise models

### 🔹 Deep Learning
- BERT / SpanBERT / PhoBERT → state-of-the-art  
- Fine-tune trên dataset: OntoNotes, WikiCoref, etc.

---

## 5. Code Python – SpaCy + NeuralCoref

```python
import spacy
import neuralcoref

nlp = spacy.load("en_core_web_sm")
neuralcoref.add_to_pipe(nlp)

text = "Nguyễn Văn A đi học. Anh ấy thích AI."
doc = nlp(text)

print(doc._.coref_clusters)
Output:
[Nguyễn Văn A: [Nguyễn Văn A, Anh ấy]]
coref_clusters → list các cluster của entity
