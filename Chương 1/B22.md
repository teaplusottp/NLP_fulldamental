# 📘 Chương 2 - Bài 22: Sentiment Analysis trong NLP

---

## 1. Sentiment Analysis là gì?

**Sentiment Analysis** = xác định **cảm xúc / thái độ** của văn bản.  

- Task phổ biến:  
  - Positive (tích cực)  
  - Negative (tiêu cực)  
  - Neutral (trung lập)  

---

## 2. Ứng dụng Sentiment Analysis

- Phân tích review sản phẩm / dịch vụ  
- Social media monitoring (Twitter, Facebook)  
- Customer feedback / Support ticket  
- Market research  

---

## 3. Types of Sentiment Analysis

### 🔹 3.1 Document-level
- Phân loại toàn bộ văn bản  
- Ví dụ: review 1 sao → negative  

### 🔹 3.2 Sentence-level
- Phân tích từng câu riêng lẻ  
- Ví dụ: "Sản phẩm tốt, nhưng giao hàng chậm" → mixed  

### 🔹 3.3 Aspect-based
- Phân tích sentiment theo **khía cạnh cụ thể**  
- Ví dụ: sản phẩm tốt → positive, giao hàng chậm → negative  

---

## 4. Feature Engineering cho Sentiment

- Từ vựng + BoW / TF-IDF  
- N-grams (bigram, trigram)  
- POS tags (tăng weight cho ADJ / ADV)  
- Emoji / punctuation features  

---

## 5. Phương pháp

### 🔹 5.1 Rule-based
- Dùng từ điển cảm xúc (positive/negative words)  
- Ưu: nhanh, dễ implement  
- Nhược: không bắt được ngữ cảnh  

### 🔹 5.2 Machine Learning
- Sử dụng features BoW / TF-IDF  
- Classifiers: Logistic Regression, SVM, Random Forest  

### 🔹 5.3 Deep Learning
- LSTM / BiLSTM / GRU → capture sequence  
- Transformer (BERT, PhoBERT) → state-of-the-art  

---

## 6. Code Python – ML-based (Scikit-learn)

```python
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import MultinomialNB

# Sample dataset
texts = ["Tôi thích sản phẩm này", "Giao hàng tệ quá", "Chất lượng ok"]
labels = ["positive", "negative", "neutral"]

# Vectorize
vectorizer = CountVectorizer()
X = vectorizer.fit_transform(texts)

# Split
X_train, X_test, y_train, y_test = train_test_split(X, labels, test_size=0.33)

# Train model
clf = MultinomialNB()
clf.fit(X_train, y_train)

# Predict
print(clf.predict(vectorizer.transform(["Sản phẩm tuyệt vời"])))
Output:
['positive']
7. Code Python – Transformer-based (BERT)
from transformers import pipeline

classifier = pipeline("sentiment-analysis", model="nlptown/bert-base-multilingual-uncased-sentiment")

text = "Sản phẩm tốt nhưng giao hàng chậm"
result = classifier(text)
print(result)
Output: star rating hoặc label positive/negative/neutral
