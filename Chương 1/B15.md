# 📘 Chương 1 - Bài 15: Stopwords & Vocabulary trong NLP

---

## 1. Stopwords là gì?

**Stopwords** = các từ phổ biến nhưng **ít mang ý nghĩa phân loại**, thường loại bỏ khi làm NLP.

### Ví dụ:

- tiếng Việt: "và", "là", "của", "có", "nhưng"  
- tiếng Anh: "the", "is", "and", "of", "a"

---

## 2. Tại sao cần loại bỏ stopwords?

- Giảm **noise**
- Giảm **kích thước vector**
- Tăng **hiệu quả model**
- Giữ lại từ quan trọng

---

## 3. Khi nào không loại bỏ stopwords?

- Task: **Sentiment Analysis**
    - từ như “không” → quan trọng
- Task: **Chatbot / LLM**
    - giữ nguyên text → preserve context

---

## 4. Code Python – loại bỏ stopwords

```python
stop_words = ["tôi", "là", "và", "của"]

text = "Tôi thích học AI và học machine learning"
tokens = text.lower().split()

filtered = [w for w in tokens if w not in stop_words]
print(filtered)
```
Output:
['thích', 'học', 'ai', 'học', 'machine', 'learning']
## 5. Vocabulary là gì?

Vocabulary = tập hợp từ duy nhất xuất hiện trong corpus/dataset

Dùng để:
Vector hóa BoW
TF-IDF
Word embeddings
## 6. Tạo Vocabulary từ corpus
corpus = [
    "tôi thích học AI",
    "tôi thích ăn cơm",
]

vocab = set()
for doc in corpus:
    vocab.update(doc.lower().split())

print(vocab)
Output:
{'tôi', 'thích', 'học', 'ai', 'ăn', 'cơm'}
## 7. Size vocabulary
Quá lớn → vector sparse → tốn bộ nhớ
Quá nhỏ → mất nghĩa → underfitting
Cách xử lý:
Giới hạn top-k frequent words
Loại bỏ low-frequency words
## 8. Stopwords + Vocabulary kết hợp
Loại stopwords → giảm noise
Tạo vocabulary → chuẩn hóa input
Vector hóa BoW / TF-IDF
