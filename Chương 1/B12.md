# 📘 Chương 1 - Bài 12: Regular Expression (Regex) trong NLP

---

## 1. Regex là gì?

**Regular Expression (Regex)** là cách dùng pattern để tìm kiếm và xử lý text.

👉 Hiểu đơn giản:
- tìm
- thay thế
- lọc text

---

## 2. Ví dụ cơ bản

### 🔹 Tìm số


"tôi có 2 con mèo và 10 con chó"


Regex:

\d+


👉 Match:

2, 10


---

## 3. Các ký hiệu quan trọng

| Ký hiệu | Ý nghĩa |
|--------|--------|
| `.` | bất kỳ ký tự |
| `\d` | số |
| `\w` | chữ + số |
| `\s` | khoảng trắng |
| `+` | 1 hoặc nhiều |
| `*` | 0 hoặc nhiều |
| `?` | 0 hoặc 1 |

---

## 4. Ví dụ thực tế

---

### 🔹 Remove punctuation

```python
import re

text = "Hello!!! NLP??"

clean = re.sub(r'[^\w\s]', '', text)

print(clean)

```
👉 Output:

Hello NLP
🔹 Remove extra spaces
re.sub(r'\s+', ' ', text)
🔹 Extract email
text = "Email tôi là abc@gmail.com"

emails = re.findall(r'\S+@\S+', text)

### 6. Pattern nâng cao
🔹 Bắt đầu / kết thúc
Ký hiệu	Ý nghĩa
^	bắt đầu
$	kết thúc
Ví dụ:
^hello

→ chỉ match nếu bắt đầu bằng "hello"

### 7. Ví dụ NLP thực tế
Input bẩn:
"Code AI!!!   mệt vl 😭😭 123"
Normalize:
text = text.lower()
text = re.sub(r'[^\w\s]', '', text)
text = re.sub(r'\d+', '', text)
text = re.sub(r'\s+', ' ', text).strip()
Output:
"code ai mệt vl"
