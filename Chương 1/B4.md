# 📘 Chương 1 - Bài 4: Stopwords & Filtering

---

## 1. Stopwords là gì?

**Stopwords** là những từ xuất hiện rất thường xuyên nhưng **không mang nhiều ý nghĩa quan trọng**.

👉 Ví dụ (tiếng Anh):
- the, is, at, on, and

👉 Ví dụ (tiếng Việt):
- là, thì, mà, và, của, một, những

---

## 2. Tại sao phải remove stopwords?

Ví dụ:
```
"Tôi rất thích học AI"
```

→ Sau khi bỏ stopwords:
```
["thích", "học", "AI"]
```

👉 Model sẽ tập trung vào **từ quan trọng**

---

## 3. Khi nào KHÔNG nên xóa stopwords?

❗ Không phải lúc nào cũng xóa:

### 🔹 Sentiment Analysis
```
"không thích"
```

Nếu bỏ "không" → sai nghĩa ❌

---

### 🔹 Chatbot / LLM
- Cần giữ ngữ cảnh tự nhiên

---

## 4. Các cách xử lý stopwords

---

### 🔹 4.1 Dùng list có sẵn

Python (NLTK):

```python
from nltk.corpus import stopwords

stop_words = set(stopwords.words('english'))
```

---

### 🔹 4.2 Tự định nghĩa

```python
stop_words = ["là", "thì", "mà", "và"]
```

---

### 🔹 4.3 Custom theo task

👉 Giữ lại:
- "không"
- "rất"

---

## 5. Ví dụ code Python

```python
text = "Tôi rất thích học AI"

stop_words = ["tôi", "rất"]

tokens = text.lower().split()

filtered = [word for word in tokens if word not in stop_words]

print(filtered)
```

Output:
```
["thích", "học", "ai"]
```

---

## 6. Filtering nâng cao

---

### 🔹 Remove short words

```python
[word for word in tokens if len(word) > 2]
```

---

### 🔹 Remove numbers

```python
[word for word in tokens if not word.isdigit()]
```

---

### 🔹 Remove rare words

- Xuất hiện quá ít trong dataset
