## 1. Skip-gram là gì?

**Skip-gram** dự đoán các **context words** từ từ **trung tâm**.

```
Center word: "cat"
              ↓
Predict:  ["The", "sat", "on", "mat"]   (window=2)
```

---

## 2. Architecture

```
Input:
  one-hot("cat") [V]
        ↓
  W_in [V×N] → hidden [N] (embedding của "cat")
        ↓
  W_out [N×V] → [V] → softmax → P(wc | "cat")

Train để maximize P("The"|"cat") * P("sat"|"cat") * ...
```

---

## 3. Training Pairs Skip-gram

```python
corpus = "The cat sat on the mat".split()
window = 2

def create_skipgram_pairs(corpus, window):
    pairs = []
    for i, center in enumerate(corpus):
        for j in range(max(0, i-window), min(len(corpus), i+window+1)):
            if i != j:
                pairs.append((center, corpus[j]))
    return pairs

pairs = create_skipgram_pairs(corpus, window)
print("Skip-gram Training Pairs (center, context):")
for center, context in pairs:
    print(f"  ({center}, {context})")
```

## 4. Vấn đề: Softmax quá chậm

Với vocabulary V = 50,000:

```
P(word | center) = exp(v_word · v_center) / Σ exp(v_w · v_center)
                                              ↑
                              Phải tính 50,000 lần → chậm!
```

## 5. Negative Sampling

**Key insight:** Thay vì predict toàn bộ vocabulary, chỉ **distinguish positive từ negative samples**.

```
Positive: (cat, sat) → P = cao
Negative: (cat, jupiter), (cat, democracy), (cat, xyz) → P = thấp
```

**Objective:**

$$J = \log \sigma(v_{context} \cdot v_{center}) + \sum_{k=1}^{K} \mathbb{E}_{w_k \sim P_n(w)}[\log \sigma(-v_{w_k} \cdot v_{center})]$$

Chỉ update **K+1** vectors thay vì **V** vectors mỗi step!

## 6. Chọn Negative Samples

```python
import numpy as np
from collections import Counter

# Negative sampling theo unigram distribution^(3/4)
# Từ phổ biến hơn → nhiều khả năng là negative sample hơn

corpus_flat = "the cat sat on the mat the dog ran the cat is quick".split()
word_freq = Counter(corpus_flat)

# Tính sampling probability
def compute_sampling_probs(word_freq, power=0.75):
    """P(w) ∝ freq(w)^0.75"""
    freq_power = {w: f**power for w, f in word_freq.items()}
    total = sum(freq_power.values())
    return {w: f/total for w, f in freq_power.items()}

probs = compute_sampling_probs(word_freq)
print("Sampling probabilities:")
for w, p in sorted(probs.items(), key=lambda x: x[1], reverse=True):
    print(f"  {w}: {p:.4f}")

# Sample negative words
vocab = list(probs.keys())
weights = [probs[w] for w in vocab]

def get_negative_samples(target, k, vocab, weights):
    """Lấy k negative samples (không phải target)"""
    samples = []
    while len(samples) < k:
        sampled = np.random.choice(vocab, p=weights)
        if sampled != target:
            samples.append(sampled)
    return samples

neg_samples = get_negative_samples("cat", k=5, vocab=vocab, weights=weights)
print(f"\nNegative samples for 'cat': {neg_samples}")
```

## 7. Training Skip-gram với Gensim

```python
from gensim.models import Word2Vec

sentences = [
    "the cat sat on the mat".split(),
    "the dog ran on the grass".split(),
    "cats are friendly animals pets".split(),
    "dogs are loyal intelligent animals".split(),
    "animals need food water shelter".split(),
    "cat and dog are common pets people keep".split(),
    "the rat sat beside the cat on mat".split()
]

# Skip-gram: sg=1
sg_model = Word2Vec(
    sentences=sentences,
    vector_size=100,
    window=5,
    min_count=1,
    sg=1,              # Skip-gram
    negative=5,        # 5 negative samples
    ns_exponent=0.75,  # Power for negative sampling distribution
    workers=2,
    epochs=200,
    seed=42
)

print("Skip-gram trained!")
print(f"Vocab: {len(sg_model.wv)} words")

# Kiểm tra semantic properties
print("\n--- Semantic Similarity ---")
print(f"cat vs dog: {sg_model.wv.similarity('cat', 'dog'):.4f}")
print(f"cat vs mat: {sg_model.wv.similarity('cat', 'mat'):.4f}")

print("\nMost similar to 'cat':")
for word, score in sg_model.wv.most_similar('cat', topn=5):
    print(f"  {word}: {score:.4f}")
```

## 8. Tại sao Skip-gram tốt hơn CBOW?

```python
# Thực nghiệm trên Mikolov et al. 2013:
# Skip-gram task: predict nhiều context từ 1 center
# → Nhiều training examples hơn CBOW
# → Từ hiếm (rare words): skip-gram học tốt hơn

rare_word_examples = {
    "CBOW approach": "average context → predict center (rare words bị dilute)",
    "Skip-gram approach": "each center → predict each context (rare center word vẫn được train nhiều)"
}

for approach, desc in rare_word_examples.items():
    print(f"\n{approach}:")
    print(f"  {desc}")
```

## 9. Subsampling Frequent Words

Từ phổ biến như "the", "is" không mang nhiều semantic:

```python
# Trong Gensim: sample param
model_with_subsampling = Word2Vec(
    sentences=sentences,
    vector_size=100,
    window=5,
    min_count=1,
    sg=1,
    sample=1e-3,   # Subsampling threshold
                   # từ có freq > t → bị downsample
    seed=42
)
```

$$P(\text{discard } w) = 1 - \sqrt{\frac{t}{f(w)}}$$

Từ "the" (freq rất cao) → bị discard nhiều, không overwhelm training.

---

## 10. Tóm tắt

- Skip-gram: **center → predict context**
- Tốt hơn CBOW với **rare words** và **small corpus**
- **Negative Sampling**: speed up training O(V) → O(K)
- K = 5-20 negative samples thường đủ
- **Subsampling**: loại frequent words → better embeddings
