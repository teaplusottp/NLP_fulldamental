## 2. Vấn đề 1: Semantic Blindness

```python
# BoW xem "good" và "great" là hai từ HOÀN TOÀN KHÁC nhau
# Cosine similarity = 0 nếu không có từ chung!

from sklearn.feature_extraction.text import CountVectorizer
from sklearn.metrics.pairwise import cosine_similarity

texts = ["this movie is good", "this film is great"]
vec = CountVectorizer()
X = vec.fit_transform(texts)
sim = cosine_similarity(X[0], X[1])[0][0]
print(f"'good' vs 'great' similarity (BoW): {sim:.4f}")
# → 0.5000 (chỉ vì có "this" và "is" chung, không phải vì "good" ≈ "great")

# Nhưng trong thực tế: good ≈ great ≈ excellent ≈ wonderful
# Word Embedding sẽ capture điều này!
```

## 3. Vấn đề 2: Không có Context


## 4. Vấn đề 3: Similarity không có nghĩa

```python
from sklearn.metrics.pairwise import cosine_similarity
import numpy as np

texts2 = [
    "The cat sat on the mat",
    "A feline rested on the rug",   # same meaning, different words
    "The stock market crashed today"  # completely different
]

vec2 = CountVectorizer()
X2 = vec2.fit_transform(texts2)
sims = cosine_similarity(X2)
print("\nBow similarities:")
print(f"  cat/mat vs feline/rug: {sims[0,1]:.4f}")  # should be HIGH
print(f"  cat/mat vs stock market: {sims[0,2]:.4f}")  # should be LOW

# Kết quả: "cat/mat" và "feline/rug" có similarity gần 0!
# Mặc dù nghĩa gần nhau.
```

## 5. Vấn đề 4: Sparse & High-dimensional

```python
# Vocabulary = 50,000 từ
# Mỗi document = vector 50,000 chiều
# Mỗi document chỉ có ~50-100 từ khác nhau
# → vector 99.9% là zero!

# Sparse = lãng phí memory, tính toán chậm
# High-dimensional = curse of dimensionality với KNN
```

---

## 6. Vấn đề 5: OOV (Out-of-Vocabulary)

```python
# Training vocab: ["cat", "dog", "fish", ...]
# Test sentence: "The labrador retriever is friendly"
# → "labrador", "retriever" không có trong vocab
# → Hoàn toàn mất mát thông tin này!

# Word Embeddings fix OOV bằng subword models (FastText, BPE)
```

---

## 7. Word Embeddings: Giải pháp

**Word Embeddings** ánh xạ mỗi từ thành **dense vector** trong không gian liên tục:

```
good    → [0.3, 0.8, -0.1, 0.5, ...]  ← 300 dimensions
great   → [0.3, 0.7, -0.1, 0.6, ...]  ← similar!
terrible→ [-0.4, -0.7, 0.2, -0.5, ...]
```

**Key properties:**
- **Dense**: không có zero, compact hơn
- **Semantic**: từ tương đồng có vectors gần nhau
- **Analogical**: king - man + woman ≈ queen

---

## 8. So sánh BoW vs Embedding

| | BoW/TF-IDF | Word Embedding |
|-|------------|----------------|
| Dimension | 50,000+ | 50-300 |
| Sparse | Rất sparse | Dense |
| Semantic | Không | Có |
| Context | Không | Có (contextual emb.) |
| OOV | Thất bại | Subword models |
| Compute | Nhanh | Chậm hơn |

## 9. Preview: Chương 3 sẽ học gì?

```
Word2Vec  → Học embedding từ co-occurrence trong corpus
GloVe     → Global vectors từ word co-occurrence matrix
FastText  → Subword embeddings (xử lý OOV)
              ↓
Contextual Embeddings (Chương 5):
ELMo, BERT → "bank" trong ngữ cảnh khác nhau → vector khác nhau
```
