# 📙 Chương 3 - Bài 14: Contextual vs Static Embedding

## 1. Static Embedding là gì?

**Static (non-contextual) embedding**: mỗi từ có đúng **một vector cố định**, bất kể ngữ cảnh.

```
"bank" → [0.2, -0.5, 0.8, ...]  ← luôn là vector này, dù trong câu nào
```

Ví dụ: **Word2Vec, GloVe, FastText**

---

## 2. Vấn đề với Static Embedding: Polysemy

```python
# Từ "bank" có nhiều nghĩa:
sentences = [
    "I went to the bank to deposit money.",          # bank = ngân hàng
    "He sat on the river bank and fished.",          # bank = bờ sông
    "The plane banked sharply to the left.",         # bank = nghiêng
]

# Với static embedding:
# "bank" → CÙNG một vector cho cả 3 câu
# Model không phân biệt được nghĩa khác nhau
```

---

## 3. Contextual Embedding là gì?

**Contextual embedding**: vector của một từ **thay đổi theo ngữ cảnh** câu.

```
"bank" trong "deposit money at bank" → vector_bank_1
"bank" trong "sit on river bank"     → vector_bank_2
"bank" trong "plane banked sharply"  → vector_bank_3
```

Ví dụ: **ELMo, BERT, GPT, RoBERTa**

---

## 4. ELMo: tiên phong Contextual Embedding

```python
# ELMo = Embeddings from Language Models (AllenNLP, 2018)
# Dùng Bi-LSTM được train trên LM task
# Mỗi từ = weighted sum của 3 layers: character, lower LSTM, upper LSTM

# Demo API (cần allennlp hoặc tensorflow_hub)
try:
    import tensorflow_hub as hub
    import tensorflow as tf
    
    elmo = hub.load("https://tfhub.dev/google/elmo/3")
    
    sentences = [
        "I went to the bank to deposit money",
        "He sat on the river bank"
    ]
    
    embeddings = elmo(sentences, signature="default", as_dict=True)["elmo"]
    print(f"ELMo shape: {embeddings.shape}")  # (2, seq_len, 1024)
    
except ImportError:
    print("tensorflow_hub not available → demo schematic:")
    print("""
ELMo Architecture:
  Input: ['I', 'went', 'to', 'the', 'bank', ...]
  
  char_CNN(I) → char_CNN(bank)
      ↓                 ↓  
  forward LSTM →→→ [h_1, h_2, ..., h_bank]
  backward LSTM ←←← [h_1, h_2, ..., h_bank]
      ↓                 ↓
  ELMo_embed(bank) = γ(w0 * char_emb + w1 * LSTM1 + w2 * LSTM2)
  → Khác nhau hoàn toàn cho hai câu có "bank"
""")
```

## 5. BERT: Contextual Embedding với Transformer

```python
from transformers import BertTokenizer, BertModel
import torch
import numpy as np
from sklearn.metrics.pairwise import cosine_similarity

# Load pre-trained BERT
try:
    tokenizer = BertTokenizer.from_pretrained('bert-base-uncased')
    model = BertModel.from_pretrained('bert-base-uncased')
    model.eval()

    def get_contextual_embedding(sentence: str, target_word: str):
        """Lấy embedding của target_word TRONG ngữ cảnh câu"""
        inputs = tokenizer(sentence, return_tensors='pt')
        tokens = tokenizer.convert_ids_to_tokens(inputs['input_ids'][0])
        
        with torch.no_grad():
            outputs = model(**inputs)
        
        # Last hidden state
        hidden = outputs.last_hidden_state[0]  # (seq_len, 768)
        
        # Tìm vị trí target word
        target_lower = target_word.lower()
        target_idx = None
        for i, tok in enumerate(tokens):
            if tok == target_lower:
                target_idx = i
                break
        
        if target_idx is None:
            return None, tokens
        
        return hidden[target_idx].numpy(), tokens

    # Test với "bank"
    sentences_bank = [
        "I went to the bank to deposit money",
        "He sat on the river bank and fished",
    ]

    embeddings_bank = []
    for sent in sentences_bank:
        emb, tokens = get_contextual_embedding(sent, "bank")
        if emb is not None:
            embeddings_bank.append(emb)
            print(f"Tokens: {tokens}")
            print(f"  'bank' embedding norm: {np.linalg.norm(emb):.4f}")

    if len(embeddings_bank) == 2:
        sim = cosine_similarity([embeddings_bank[0]], [embeddings_bank[1]])[0][0]
        print(f"\nCosine similarity 'bank'(finance) vs 'bank'(river) = {sim:.4f}")
        print("→ Different context → different embedding ✅" if sim < 0.95 else "→ Similar (small model)")

except Exception as e:
    print(f"BERT demo: {e}")
    print("Schematic: BERT gives different 768-dim vectors for same word in different contexts")
```

## 6. So sánh trực quan

```python
print("""
=== STATIC vs CONTEXTUAL EMBEDDING ===

Static (Word2Vec, GloVe):
  "I sat on the bank"     → bank = [0.3, -0.1, 0.8, ...]
  "I went to the bank"    → bank = [0.3, -0.1, 0.8, ...]  ← SAME ❌

Contextual (BERT):
  "I sat on the river bank" → bank = [0.1, 0.9, -0.2, ...]  (river-related)
  "I went to the bank"      → bank = [0.7, -0.3, 0.5, ...]  (finance-related)
  → DIFFERENT ✅

Why it matters:
  WSD (Word Sense Disambiguation) cần contextual
  Sentiment Analysis: "fine" trong "I'm not fine" vs "fine wine"
  NER: "Apple" công ty vs "apple" quả
""")
```

| Training cost | Thấp | Trung bình | Cao |
| Inference speed | Nhanh | Trung bình | Chậm |
| Memory | Nhỏ | Trung bình | Lớn |
| Quality | Tốt | Tốt hơn | Tốt nhất |

---

## 8. Tóm tắt

- **Static embedding**: nhanh, nhẹ, nhưng không xử lý được polysemy
- **Contextual embedding**: mạnh hơn, xử lý đa nghĩa, train costs cao
- ELMo (BiLSTM) → BERT (Transformer encoder) → hiện đại nhất
- Trong production: chọn theo tradeoff tốc độ vs chất lượng
