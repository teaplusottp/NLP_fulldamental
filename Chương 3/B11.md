# 📙 Chương 3 - Bài 11: Trực quan hóa Embedding Space (PCA & t-SNE)

## 1. Tại sao cần Visualization?

Embedding là vector 100–300 chiều → không thể nhìn trực tiếp. Visualization giúp:
- Kiểm tra xem model có học được semantic clusters không
- Debug khi model hoạt động kém
- Giải thích embedding cho stakeholders

---

## 2. Hai kỹ thuật chính

### PCA (Principal Component Analysis)
- **Linear**, nhanh, deterministic
- Giữ maximum variance
- Tốt cho overview, reproducible

### t-SNE (t-Distributed Stochastic Neighbor Embedding)
- **Non-linear**, chậm hơn
- Giữ cấu trúc local (clusters)
- Tốt cho thấy groupings, nhưng axes không có nghĩa

---

## 3. Code: PCA Visualization

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.decomposition import PCA
from gensim.models import Word2Vec
import re

# Train model
corpus = [
    ["king", "queen", "prince", "princess", "royal", "castle"],
    ["man", "woman", "boy", "girl", "person", "human"],
    ["cat", "dog", "rabbit", "hamster", "pet", "animal"],
    ["car", "bus", "train", "plane", "boat", "transport"],
    ["python", "java", "ruby", "code", "program", "software"],
    ["happy", "sad", "angry", "fearful", "joyful", "emotion"],
    ["apple", "banana", "orange", "grape", "mango", "fruit"],
    ["doctor", "nurse", "patient", "hospital", "medicine", "health"],
]

model = Word2Vec(corpus, vector_size=50, window=5, min_count=1, sg=1, epochs=500, seed=42)

# Nhóm từ theo category
groups = {
    'Royals': ['king', 'queen', 'prince', 'princess'],
    'People': ['man', 'woman', 'boy', 'girl'],
    'Animals': ['cat', 'dog', 'rabbit', 'hamster'],
    'Transport': ['car', 'bus', 'train', 'plane'],
    'Tech': ['python', 'java', 'ruby', 'code'],
    'Emotion': ['happy', 'sad', 'angry', 'joyful'],
    'Fruit': ['apple', 'banana', 'orange', 'grape'],
    'Medical': ['doctor', 'nurse', 'patient', 'hospital'],
}

words_all, labels_all = [], []
for group, words in groups.items():
    for w in words:
        if w in model.wv:
            words_all.append(w)
            labels_all.append(group)

X = np.array([model.wv[w] for w in words_all])

# PCA 2D
pca = PCA(n_components=2, random_state=42)
X_pca = pca.fit_transform(X)

colors = ['#e74c3c','#3498db','#2ecc71','#f39c12','#9b59b6','#1abc9c','#e67e22','#34495e']
group_list = list(groups.keys())

plt.figure(figsize=(12, 8))
for word, label, (x, y) in zip(words_all, labels_all, X_pca):
    c = colors[group_list.index(label)]
    plt.scatter(x, y, c=c, s=100, alpha=0.8, zorder=3)
    plt.annotate(word, (x, y), xytext=(3, 3), textcoords='offset points', fontsize=9)

for group, c in zip(group_list, colors):
    plt.scatter([], [], c=c, label=group, s=80)

plt.legend(loc='best', fontsize=9)
plt.title(f"PCA 2D Visualization\n(Explained variance: {pca.explained_variance_ratio_.sum():.2%})")
plt.xlabel(f"PC1 ({pca.explained_variance_ratio_[0]:.1%})")
plt.ylabel(f"PC2 ({pca.explained_variance_ratio_[1]:.1%})")
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.show()
```

## 4. Code: t-SNE Visualization

```python
from sklearn.manifold import TSNE

# t-SNE (chạy nhiều chiều hơn)
X_full = np.array([model.wv[w] for w in words_all])

tsne = TSNE(n_components=2, perplexity=8, random_state=42,
            n_iter=1000, learning_rate='auto', init='random')
X_tsne = tsne.fit_transform(X_full)

plt.figure(figsize=(12, 8))
for word, label, (x, y) in zip(words_all, labels_all, X_tsne):
    c = colors[group_list.index(label)]
    plt.scatter(x, y, c=c, s=100, alpha=0.8, zorder=3)
    plt.annotate(word, (x, y), xytext=(3, 3), textcoords='offset points', fontsize=9)

for group, c in zip(group_list, colors):
    plt.scatter([], [], c=c, label=group, s=80)

plt.legend(loc='best', fontsize=9)
plt.title("t-SNE 2D Visualization (perplexity=8)")
plt.xlabel("t-SNE dim 1")
plt.ylabel("t-SNE dim 2")
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.show()
```

## 5. Heatmap Cosine Similarity

```python
# Heatmap similarity giữa các từ
from sklearn.metrics.pairwise import cosine_similarity
import seaborn as sns

sample_words = ['king', 'queen', 'man', 'woman', 'cat', 'dog', 'python', 'java']
sample_vecs = np.array([model.wv[w] for w in sample_words if w in model.wv])
valid_words = [w for w in sample_words if w in model.wv]
sim_matrix = cosine_similarity(sample_vecs)

plt.figure(figsize=(8, 6))
sns.heatmap(sim_matrix, xticklabels=valid_words, yticklabels=valid_words,
            annot=True, fmt='.2f', cmap='RdYlGn', center=0, vmin=-1, vmax=1)
plt.title("Cosine Similarity Heatmap")
plt.tight_layout()
plt.show()
```

## 6. PCA vs t-SNE

| Tiêu chí | PCA | t-SNE |
|---------|-----|-------|
| Linear? | ✅ | ❌ |
| Tốc độ | Nhanh | Chậm (O(n²)) |
| Reproducible | ✅ | Chỉ với seed |
| Axes có nghĩa | ✅ | ❌ |
| Show local structure | Trung bình | ✅ xuất sắc |
| Khi nào dùng | Overview nhanh | Explore clusters |

---

## 7. Tóm tắt

- **PCA**: nhanh, linear, axes có nghĩa, dùng để debug nhanh
- **t-SNE**: tốt hơn để thấy clusters, nhưng cần chọn `perplexity` cẩn thận
- Kết hợp 2: PCA giảm chiều → t-SNE tiếp (PCA 50D → t-SNE 2D)
- Visualization là công cụ không thể thiếu khi làm việc với embeddings
