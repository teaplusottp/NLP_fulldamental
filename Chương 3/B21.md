# 📙 Chương 3 - Bài 21: Fine-tuning Embedding (Domain Adaptation)

## 1. Fine-tuning Embedding là gì?

**Pre-trained embeddings** được train trên general corpus (Wikipedia, News).  
**Domain-specific** text (y tế, pháp lý, tài chính) có vocabulary & style riêng.

**Fine-tuning**: tiếp tục train pre-trained embedding trên domain corpus.

---

## 2. Khi nào cần Fine-tuning?

```python
from gensim.models import Word2Vec
import numpy as np

# General corpus (simulating Wikipedia-like)
general_corpus = [
    ["bank", "money", "finance", "deposit", "loan"],
    ["bank", "river", "shore", "water", "flood"],
    ["cell", "biology", "organism", "membrane"],
    ["cell", "phone", "mobile", "call", "number"],
]

general_model = Word2Vec(general_corpus, vector_size=50, min_count=1,
                         sg=1, epochs=300, seed=42)

# Kiểm tra similarity trong domain y tế vs chung
medical_words = ["cell", "membrane", "organism"]
finance_words = ["bank", "deposit", "loan"]

print("=== General embedding similarity ===")
for w1, w2 in [("cell", "membrane"), ("cell", "phone"), ("bank", "money"), ("bank", "river")]:
    if w1 in general_model.wv and w2 in general_model.wv:
        sim = general_model.wv.similarity(w1, w2)
        print(f"  '{w1}' vs '{w2}': {sim:.4f}")

print("\n→ General model MIXES medical and financial meaning of 'cell' and 'bank'")
```

## 3. Fine-tuning Word2Vec

```python
# Step 1: Load pre-trained model
# (Dùng general_model đã train ở trên làm base)

# Step 2: Prepare domain corpus
medical_domain_corpus = [
    ["cell", "membrane", "nucleus", "cytoplasm", "organelle"],
    ["cell", "division", "mitosis", "meiosis", "chromosome"],
    ["protein", "enzyme", "receptor", "membrane", "cell"],
    ["cancer", "cell", "tumor", "malignant", "oncology"],
    ["stem", "cell", "bone", "marrow", "blood"],
    ["immune", "cell", "lymphocyte", "antibody", "pathogen"],
    ["neural", "cell", "neuron", "synapse", "brain", "axon"],
] * 20  # repeat to simulate larger domain corpus

# Step 3: Fine-tune = build vocab + update vectors
import copy
finetuned_model = copy.deepcopy(general_model)

# Update vocabulary với domain words
finetuned_model.build_vocab(medical_domain_corpus, update=True)

# Continue training on domain corpus
finetuned_model.train(
    medical_domain_corpus,
    total_examples=finetuned_model.corpus_count,
    epochs=200
)

print("=== Fine-tuned embedding similarity (medical domain) ===")
test_pairs = [
    ("cell", "membrane"),    # medical meaning
    ("cell", "nucleus"),     # medical meaning
    ("cell", "phone"),       # now should be further apart
    ("bank", "money"),       # unchanged
]
for w1, w2 in test_pairs:
    if w1 in finetuned_model.wv and w2 in finetuned_model.wv:
        sim = finetuned_model.wv.similarity(w1, w2)
        print(f"  '{w1}' vs '{w2}': {sim:.4f}")
```

## 4. Fine-tuning GloVe/FastText

```python
from gensim.models import FastText

# FastText dễ fine-tune hơn nhờ subword
base_ft = FastText(
    sentences=[["natural", "language", "processing", "machine", "learning"]],
    vector_size=50, min_count=1, sg=1, epochs=100, seed=42
)

# Domain corpus
domain_corpus = [
    ["diagnostic", "clinical", "pathology", "prognosis", "treatment"],
    ["electrocardiogram", "echocardiography", "radiography", "diagnosis"],
    ["hypertension", "coronary", "artery", "cardiovascular", "therapy"],
] * 10

# Fine-tune FastText
base_ft.build_vocab(domain_corpus, update=True)
base_ft.train(domain_corpus, total_examples=len(domain_corpus), epochs=100)

# Test medical OOV với subword
medical_oov = ["electrocardiographic", "pathological", "hypertensive"]
print("\n=== FastText domain fine-tuning + OOV ===")
for word in medical_oov:
    vec = base_ft.wv[word]  # Always works (subword)
    print(f"  '{word}': norm={np.linalg.norm(vec):.3f} ✅")
```

## 5. Catastrophic Forgetting

```python
# Vấn đề: khi fine-tune quá nhiều trên domain,
# model có thể "quên" general knowledge

print("""
Catastrophic Forgetting:
  - Fine-tune 100 epochs trên medical corpus
  - "car" (không có trong medical) → embedding bị drift
  - "bank" (có nhưng không dùng nhiều) → embedding thay đổi

Solutions:
  1. Learning rate thấp khi fine-tuning
  2. Regularization (L2 penalty toward original params)
  3. Mix general + domain corpus
  4. Freeze some layers (for neural models like BERT)
""")

# Demo: Mix general + domain corpus
mixed_corpus = general_corpus * 5 + medical_domain_corpus
mixed_model = Word2Vec(mixed_corpus, vector_size=50, min_count=1,
                       sg=1, epochs=200, seed=42)

print("=== Mixed corpus embedding ===")
for (w1, w2) in [("cell", "membrane"), ("cell", "phone"),
                  ("bank", "money"), ("bank", "river")]:
    if w1 in mixed_model.wv and w2 in mixed_model.wv:
        sim = mixed_model.wv.similarity(w1, w2)
        print(f"  '{w1}' vs '{w2}': {sim:.4f}")
```

## 6. Evaluation: before vs after fine-tuning

```python
def domain_eval(model, domain_pairs: list, label: str):
    print(f"\n  {label}:")
    for w1, w2, expected_high in domain_pairs:
        if w1 in model.wv and w2 in model.wv:
            sim = model.wv.similarity(w1, w2)
            status = "✅" if (sim > 0.5) == expected_high else "❌"
            print(f"    '{w1}' vs '{w2}': {sim:.4f} {status}")

medical_pairs = [
    ("cell", "membrane", True),
    ("cell", "nucleus", True),
    ("protein", "enzyme", True),
]

print("=== Domain Evaluation ===")
domain_eval(general_model, medical_pairs, "General (before fine-tune)")
domain_eval(finetuned_model, medical_pairs, "Fine-tuned (after)")
domain_eval(mixed_model, medical_pairs, "Mixed corpus")
```

## 7. Tóm tắt

| Strategy | Catastrophic Forgetting | Domain Quality | Complexity |
|---------|------------------------|---------------|-----------|
| Train from scratch | ❌ N/A | Good | High |
| Fine-tune (high LR) | High | Very Good | Low |
| Fine-tune (low LR) | Low | Good | Low |
| Mixed corpus | None | Good | Medium |
| Freeze + adapter | None | Best | High (BERT) |

- Fine-tuning word embedding thường đơn giản và hiệu quả
- Dùng low LR và mixed corpus để tránh forgetting
- Với BERT/transformers: adapter layers là gold standard
