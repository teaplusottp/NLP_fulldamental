## 1. Multilingual Embeddings là gì?

**Multilingual embedding**: map words từ nhiều ngôn ngữ vào **cùng một vector space**.

```
"cat"  (English) → [0.2, -0.5, ...]
"chat" (French)  → [0.21, -0.49, ...]  ← gần với "cat"!
"猫"   (Chinese) → [0.19, -0.51, ...]  ← cũng gần!
"mèo"  (Vietnamese) → [0.22, -0.48, ...]
```

Cho phép cross-lingual tasks: translate, zero-shot classification, etc.

## 2. Bilingual Word Embeddings

```python
import numpy as np
from sklearn.metrics.pairwise import cosine_similarity
import warnings
warnings.filterwarnings('ignore')

# Simulate bilingual word pairs (English-French)
# Trong thực tế: dùng parallel corpora

# English embeddings (pretrained)
np.random.seed(42)
dim = 50

def create_language_vectors(words: list, noise_scale=0.1) -> dict:
    """Tạo fake embeddings với semantic clusters"""
    categories = {
        'animals': ['cat', 'dog', 'bird', 'fish', 'horse', 'rabbit'],
        'transport': ['car', 'train', 'plane', 'bus', 'ship', 'bike'],
        'food': ['apple', 'bread', 'rice', 'meat', 'milk', 'egg'],
        'tech': ['computer', 'phone', 'network', 'data', 'code', 'software'],
    }
    
    centers = {cat: np.random.randn(dim) for cat in categories}
    word_vecs = {}
    
    for cat, cat_words in categories.items():
        for word in cat_words:
            if word in words:
                noise = np.random.randn(dim) * noise_scale
                word_vecs[word] = centers[cat] + noise
    
    # Words not in any category
    for word in words:
        if word not in word_vecs:
            word_vecs[word] = np.random.randn(dim) * 0.5
    
    return word_vecs, centers

en_words = ['cat', 'dog', 'bird', 'fish', 'car', 'train', 'plane',
            'apple', 'bread', 'rice', 'computer', 'phone', 'network']
fr_words = ['chat', 'chien', 'oiseau', 'poisson', 'voiture', 'train', 'avion',
            'pomme', 'pain', 'riz', 'ordinateur', 'téléphone', 'réseau']

en_vecs, en_centers = create_language_vectors(en_words)
# French = English + small rotation + noise
np.random.seed(123)
rotation = np.random.randn(dim, dim) * 0.1 + np.eye(dim)

fr_vecs_raw, _ = create_language_vectors(fr_words, noise_scale=0.15)
# Map French to similar space as English
fr_vecs = {}
for fr_word, en_word in zip(fr_words, en_words):
    # In real bilingual: learn a mapping matrix W
    fr_vecs[fr_word] = en_vecs[en_word] + np.random.randn(dim) * 0.2

print("=== Bilingual similarity check ===")
pairs = [('cat', 'chat'), ('car', 'voiture'), ('apple', 'pomme')]
for en, fr in pairs:
    sim = cosine_similarity([en_vecs[en]], [fr_vecs[fr]])[0][0]
    print(f"  '{en}' (EN) vs '{fr}' (FR): {sim:.4f}")
```

## 3. Cross-lingual Word Mapping (Procrustes)

```python
def procrustes_alignment(source_vecs: np.ndarray, target_vecs: np.ndarray) -> np.ndarray:
    """
    Tìm orthogonal matrix W sao cho source_vecs @ W ≈ target_vecs
    Giải: SVD của target_vecs.T @ source_vecs
    """
    M = target_vecs.T @ source_vecs
    U, s, Vt = np.linalg.svd(M)
    W = U @ Vt
    return W

# Seed pairs (bilingual dictionary): EN → FR
seed_pairs_en = ['cat', 'dog', 'car', 'apple', 'computer']
seed_pairs_fr = ['chat', 'chien', 'voiture', 'pomme', 'ordinateur']

# Get vectors
src_mat = np.array([fr_vecs_raw.get(fr, np.random.randn(dim)) for fr in seed_pairs_fr])
tgt_mat = np.array([en_vecs[en] for en in seed_pairs_en])

# Normalize
src_mat = src_mat / (np.linalg.norm(src_mat, axis=1, keepdims=True) + 1e-9)
tgt_mat = tgt_mat / (np.linalg.norm(tgt_mat, axis=1, keepdims=True) + 1e-9)

# Learn mapping
W = procrustes_alignment(src_mat, tgt_mat)

# Apply mapping to all French vectors
def map_to_english_space(fr_word: str, W: np.ndarray) -> np.ndarray:
    vec = fr_vecs_raw.get(fr_word, np.zeros(dim))
    return vec @ W.T

# Test: find nearest EN word for each FR word
print("\n=== Cross-lingual mapping ===")
test_fr = ['chat', 'chien', 'avion', 'pain', 'réseau']
true_en = ['cat', 'dog', 'plane', 'bread', 'network']

for fr, expected_en in zip(test_fr, true_en):
    mapped_vec = map_to_english_space(fr, W)
    
    # Find nearest EN word
    en_vocab = list(en_vecs.keys())
    en_mat = np.array([en_vecs[w] for w in en_vocab])
    sims = cosine_similarity([mapped_vec], en_mat)[0]
    predicted_en = en_vocab[np.argmax(sims)]
    
    match = "✅" if predicted_en == expected_en else f"❌ (pred: {predicted_en})"
    print(f"  '{fr}' → '{predicted_en}' (expected: '{expected_en}') {match}")
```

## 5. XLM-R (Cross-lingual Model RoBERTa)

```python
print("""
=== XLM-RoBERTa ===

XLM-R (Conneau et al. 2020):
  - RoBERTa style training on 100 languages
  - 2.5TB clean CommonCrawl data
  - SentencePiece tokenizer (250K vocab)
  - State-of-the-art cross-lingual performance
  
Nhà hơn mBERT vì:
  1. Larger pre-training corpus (100x more non-English data)
  2. Removed language embeddings (forced to share representations)
  3. SentencePiece > WordPiece for multilingual

from transformers import XLMRobertaTokenizer, XLMRobertaModel
```

## 6. Tóm tắt

| Model | Languages | Method | Best For |
|-------|----------|--------|---------|
| Bilingual W2V | 2 | Procrustes | Simple translation |
| mBERT | 104 | Joint MLM | Cross-lingual NLU |
| XLM-R | 100 | Joint MLM (more data) | Best cross-lingual |
| LaBSE | 109 | Sentence-level | Cross-lingual STS |

- Multilingual embeddings = map languages to shared space
- Zero-shot cross-lingual: train EN → apply to all languages
- XLM-R hiện là model tốt nhất cho đa ngôn ngữ
