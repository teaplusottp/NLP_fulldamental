## 1. Distributional Hypothesis là gì?

> **"You shall know a word by the company it keeps."** — J.R. Firth (1957)

**Distributional Hypothesis**: Các từ xuất hiện trong **context tương tự** thì có **nghĩa tương tự**.

---

## 2. Ví dụ trực quan

```
"She ate _____ for breakfast."   → cereal, eggs, toast, oatmeal
"She drank _____ in the morning." → coffee, tea, juice, water

Words filling "ate [blank]": food items
Words filling "drank [blank]": beverages

→ Hai blank này có context khác nhau → nghĩa khác nhau
```

## 3. Co-occurrence Matrix

**Ý tưởng cơ bản: đếm từ xuất hiện cùng nhau.**

```python
import numpy as np
from collections import defaultdict

corpus = [
    "I enjoy deep learning",
    "I enjoy NLP",
    "Deep learning is powerful",
    "NLP uses deep learning",
    "I like machine learning"
]

# Tokenize
tokens_per_sentence = [s.lower().split() for s in corpus]
all_words = list(set(w for sent in tokens_per_sentence for w in sent))
word2idx = {w: i for i, w in enumerate(sorted(all_words))}
idx2word = {i: w for w, i in word2idx.items()}

vocab_size = len(all_words)
window_size = 2

# Build co-occurrence matrix
cooc_matrix = np.zeros((vocab_size, vocab_size))

for tokens in tokens_per_sentence:
    for i, target in enumerate(tokens):
        for j in range(max(0, i-window_size), min(len(tokens), i+window_size+1)):
            if i != j:
                context = tokens[j]
                cooc_matrix[word2idx[target]][word2idx[context]] += 1

print("Vocabulary:", list(word2idx.keys()))
print(f"Co-occurrence matrix shape: {cooc_matrix.shape}")

# Xem co-occurrence của "learning"
if 'learning' in word2idx:
    idx = word2idx['learning']
    print(f"\nCo-occurrence of 'learning':")
    for j, count in enumerate(cooc_matrix[idx]):
        if count > 0:
            print(f"  {idx2word[j]}: {count:.0f}")
```

## 4. Vấn đề với Raw Co-occurrence

```python
# Từ phổ biến như "the", "is", "a" xuất hiện với gần mọi từ
# → Không discriminative

# Giải pháp: PMI (Pointwise Mutual Information)
# PMI(w, c) = log P(w,c) / (P(w) * P(c))
# PMI cao → w và c xuất hiện cùng nhau nhiều hơn ngẫu nhiên

total_count = cooc_matrix.sum()
word_counts = cooc_matrix.sum(axis=1)
context_counts = cooc_matrix.sum(axis=0)

# PPMI (Positive PMI)
ppmi_matrix = np.zeros_like(cooc_matrix)
for i in range(vocab_size):
    for j in range(vocab_size):
        if cooc_matrix[i,j] > 0:
            p_wc = cooc_matrix[i,j] / total_count
            p_w = word_counts[i] / total_count
            p_c = context_counts[j] / total_count
            if p_w > 0 and p_c > 0:
                pmi = np.log2(p_wc / (p_w * p_c))
                ppmi_matrix[i,j] = max(0, pmi)  # PPMI: negative → 0

print("\nPPMI matrix cho 'learning':")
if 'learning' in word2idx:
    idx = word2idx['learning']
    for j, val in enumerate(ppmi_matrix[idx]):
        if val > 0:
            print(f"  {idx2word[j]}: PPMI={val:.3f}")
```

## 5. Từ Co-occurrence đến Embedding

Co-occurrence matrix → áp dụng **SVD** → dense, low-dimensional embeddings:

```python
from sklearn.decomposition import TruncatedSVD

# Reduce co-occurrence matrix đến k=3 dimensions
svd = TruncatedSVD(n_components=min(3, vocab_size-1), random_state=42)
word_vectors = svd.fit_transform(ppmi_matrix)

print(f"\nWord vectors shape: {word_vectors.shape}")
print("Word vectors:")
for word, idx in word2idx.items():
    print(f"  {word:15s}: {word_vectors[idx].round(3)}")
```

## 6. Similarity từ Co-occurrence Vectors

```python
from sklearn.metrics.pairwise import cosine_similarity

def word_similarity(word1, word2, vectors, w2idx):
    if word1 not in w2idx or word2 not in w2idx:
        return None
    v1 = vectors[w2idx[word1]].reshape(1, -1)
    v2 = vectors[w2idx[word2]].reshape(1, -1)
    return cosine_similarity(v1, v2)[0][0]

pairs = [
    ('deep', 'learning'),
    ('nlp', 'learning'),
    ('enjoy', 'like'),
    ('deep', 'enjoy')
]

print("\nWord similarities:")
for w1, w2 in pairs:
    sim = word_similarity(w1, w2, word_vectors, word2idx)
    if sim is not None:
        print(f"  {w1} vs {w2}: {sim:.4f}")
```

## 7. Từ Count đến Neural (Word2Vec)

Word2Vec thay thế count-based approach bằng **neural network**:

```
Count-based: "how often do words co-occur?"
Neural (Word2Vec): "predict context words từ target word"
```

| | Count-based | Neural (Word2Vec) |
|-|-------------|-------------------|
| Speed (train) | Nhanh | Chậm hơn |
| Large corpus | Tốt | Tốt hơn |
| Analogy | Trung bình | Rất tốt |
| Memory | Ma trận lớn | Compact |

---

## 8. Tóm tắt

- **Distributional Hypothesis**: meaning từ context
- **Co-occurrence matrix**: đếm words xuất hiện cùng nhau
- **PPMI**: normalize để loại bỏ bias từ phổ biến
- **SVD**: reduce dimension → dense embeddings
- Đây là **foundation** của Word2Vec, GloVe và tất cả embeddings
