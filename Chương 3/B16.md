## 2. Intrinsic Evaluation Pitfalls

import numpy as np
from scipy.stats import spearmanr, pearsonr
from gensim.models import Word2Vec

# PITFALL 1: Human similarity ≠ NLP task similarity
# "Tiger" và "Lion" rất similar (thú hoang)
# Nhưng sentiment analysis không cần phân biệt chúng
print("""
Pitfall 1: Similarity datasets không tương quan với task

## 3. Intrinsic: Correlation Analysis

```python
# So sánh nhiều models trên cùng benchmark

def simulate_model_scores(seed=42):
    """Simulate scores của nhiều models"""
    np.random.seed(seed)
    models = {
        'Word2Vec-SGNS-100d': {'sim353': 0.74, 'simlex': 0.43, 'clf_acc': 0.88},
        'Word2Vec-CBOW-100d':  {'sim353': 0.69, 'simlex': 0.38, 'clf_acc': 0.84},
        'GloVe-100d':          {'sim353': 0.77, 'simlex': 0.41, 'clf_acc': 0.87},
        'GloVe-300d':          {'sim353': 0.81, 'simlex': 0.44, 'clf_acc': 0.91},
        'FastText-100d':       {'sim353': 0.71, 'simlex': 0.39, 'clf_acc': 0.86},
        'Random-100d':         {'sim353': 0.12, 'simlex': 0.08, 'clf_acc': 0.52},
    }
    return models

models = simulate_model_scores()

print("=== Model Comparison ===")
print(f"{'Model':25s} {'WS-353':>8} {'SimLex':>8} {'CLF Acc':>10}")
print("-" * 55)
for name, scores in models.items():
    print(f"{name:25s} {scores['sim353']:8.3f} {scores['simlex']:8.3f} {scores['clf_acc']:10.2%}")

# Correlation: intrinsic vs extrinsic
intrinsic_sim353 = [s['sim353'] for s in models.values()]
extrinsic_clf = [s['clf_acc'] for s in models.values()]
corr, p = spearmanr(intrinsic_sim353, extrinsic_clf)
print(f"\nSpearman(WS-353, CLF): ρ={corr:.4f}, p={p:.4f}")
print("→ Moderate correlation → intrinsic scores are useful but NOT perfect proxies")
```

## 4. Extrinsic: Proper Setup

```python
# Extrinsic evaluation đúng cách
from sklearn.model_selection import StratifiedKFold
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import classification_report

def proper_extrinsic_eval(embeddings_dict: dict, X_texts, y_labels):
    """
    Đánh giá nhiều embedding methods trên cùng task
    với cùng classifier và cùng splits
    """
    from sklearn.preprocessing import LabelEncoder
    le = LabelEncoder()
    y = le.fit_transform(y_labels)
    
    kf = StratifiedKFold(n_splits=3, shuffle=True, random_state=42)
    results = {}
    
    for emb_name, embed_fn in embeddings_dict.items():
        X = np.array([embed_fn(t) for t in X_texts])
        
        scores = []
        for train_idx, test_idx in kf.split(X, y):
            clf = LogisticRegression(max_iter=300, random_state=42, C=1.0)
            clf.fit(X[train_idx], y[train_idx])
            score = clf.score(X[test_idx], y[test_idx])
            scores.append(score)
        
        results[emb_name] = {
            'mean': np.mean(scores),
            'std': np.std(scores),
            'scores': scores
        }
    
    print("\n=== Extrinsic Evaluation Results ===")
    for name, r in results.items():
        print(f"  {name:20s}: {r['mean']:.3f} ± {r['std']:.3f}")
    
    return results

# Example embedding functions
def random_embed(text, dim=50):
    np.random.seed(hash(text) % 2**31)
    return np.random.randn(dim)

def length_embed(text, dim=50):
    tokens = text.split()
    base = np.zeros(dim)
    base[0] = len(tokens)
    base[1] = len(text)
    return base

# Simple text dataset
texts = ["cat dog pet animal"] * 10 + ["car bus vehicle"] * 10 + ["happy joy emotion"] * 10
labels = ["animal"] * 10 + ["vehicle"] * 10 + ["emotion"] * 10

proper_extrinsic_eval(
    {"random": random_embed, "length_feat": length_embed},
    texts, labels
)
```

## 5. Bao nhiêu % correlation?

Nghiên cứu thực (Chiu et al. 2016, Schnabel et al. 2015):

| Setup | Corr(intrinsic, extrinsic) |
|-------|--------------------------|
| Similarity → CLF | ρ ≈ 0.4–0.6 |
| Analogy → CLF | ρ ≈ 0.3–0.5 |
| Similarity → NER | ρ ≈ 0.5–0.7 |
| Best case | ρ ≈ 0.8 |

**Kết luận**: Intrinsic là proxy, không thay thế được extrinsic.

## 6. Khi nào dùng intrinsic, khi nào dùng extrinsic?

| Situation | Dùng |
|----------|------|
| Đang train, cần monitor nhanh | Intrinsic |
| Chọn embedding model cho production | Extrinsic |
| Debug embedding quality | Intrinsic + Visualization |
| Paper submission | Cả hai |
| Time constraint | Intrinsic trước, extrinsic sau |

---

## 7. Tóm tắt

- Intrinsic: nhanh, dễ implement, nhưng có bias và không luôn dự đoán được downstream performance
- Extrinsic: ground truth thực, nhưng tốn tài nguyên
- Correlation giữa hai loại là moderate (~0.4–0.7)
- Best practice: dùng intrinsic trong training loop, extrinsic cho final selection
