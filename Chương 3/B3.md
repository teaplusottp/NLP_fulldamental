## 1. Word2Vec là gì?

**Word2Vec** (Mikolov et al., 2013 - Google) là model học **word embeddings** bằng cách train neural network nhỏ để **predict context words**.

**Ý tưởng:** Thay vì đếm co-occurrence, ta train model:
- "nhìn từ trung tâm → predict surrounding words" **(Skip-gram)**
- "nhìn surrounding words → predict từ trung tâm" **(CBOW)**

---

## 2. Tại sao Word2Vec revolutionary?

Trước Word2Vec (2013):
- Embeddings từ SVD: tính toán chậm với corpus lớn
- Không scale tốt

Word2Vec:
- Train trên **hàng tỷ words** trong vài giờ
- Chất lượng embeddings xuất sắc
- Open-source → accessible cho mọi người

---

## 3. Architecture Overview

```
           Word2Vec
              │
    ┌─────────┴─────────┐
    │                   │
  CBOW               Skip-gram
(Continuous        (Predict context
 Bag of Words)      from center word)

Context → Center    Center → Context
"The ___ sat"  →  "sat" → "cat", "on"
      ↓              ↓
   [faster]       [better quality]
```

## 4. Quick Demo với Gensim

```python
# pip install gensim
from gensim.models import Word2Vec

# Corpus nhỏ để demo
sentences = [
    ["I", "love", "deep", "learning"],
    ["I", "enjoy", "machine", "learning"],
    ["NLP", "is", "amazing", "and", "fun"],
    ["deep", "learning", "is", "powerful"],
    ["I", "like", "natural", "language", "processing"],
    ["word", "embeddings", "are", "useful", "for", "NLP"],
    ["machine", "learning", "and", "deep", "learning", "are", "related"],
    ["natural", "language", "processing", "uses", "machine", "learning"],
    ["I", "study", "AI", "and", "machine", "learning"],
    ["neural", "networks", "power", "deep", "learning"]
]

# Train Word2Vec model
model = Word2Vec(
    sentences=sentences,
    vector_size=50,       # embedding dimension
    window=3,             # context window size
    min_count=1,          # minimum word frequency
    sg=1,                 # 1=Skip-gram, 0=CBOW
    workers=2,            # parallel workers
    epochs=100,           # training epochs
    seed=42
)

print(f"Vocabulary size: {len(model.wv)}")
print(f"Vector dimension: {model.wv.vector_size}")
print(f"\nVector for 'learning': {model.wv['learning'][:10]}...")
```

## 5. Word Similarity

```python
# Similarity giữa các từ
print("\nWord Similarities:")
pairs = [
    ('deep', 'machine'),
    ('learning', 'NLP'),
    ('deep', 'learning'),
    ('love', 'like')
]
for w1, w2 in pairs:
    try:
        sim = model.wv.similarity(w1, w2)
        print(f"  {w1} vs {w2}: {sim:.4f}")
    except KeyError as e:
        print(f"  {e} not in vocab")

# Most similar words
print("\nWords most similar to 'learning':")
similar = model.wv.most_similar('learning', topn=5)
for word, score in similar:
    print(f"  {word}: {score:.4f}")
```

## 7. Visualize Embeddings

```python
from sklearn.decomposition import TruncatedSVD
import matplotlib.pyplot as plt
import numpy as np

# Lấy vectors
words = list(model.wv.key_to_index.keys())
vectors = np.array([model.wv[w] for w in words])

# Reduce xuống 2D
svd = TruncatedSVD(n_components=2, random_state=42)
vectors_2d = svd.fit_transform(vectors)

plt.figure(figsize=(12, 8))
plt.scatter(vectors_2d[:, 0], vectors_2d[:, 1], alpha=0.5)

for i, word in enumerate(words):
    plt.annotate(word, (vectors_2d[i, 0], vectors_2d[i, 1]),
                 fontsize=9, ha='center', va='bottom')

plt.title("Word2Vec Embeddings (2D via SVD)")
plt.xlabel("Component 1")
plt.ylabel("Component 2")
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.savefig("word2vec_viz.png", dpi=100)
plt.show()
```

## 9. Hyperparameters quan trọng

| Param | Meaning | Typical Value |
|-------|---------|---------------|
| `vector_size` | Embedding dimension | 100-300 |
| `window` | Context window | 5 (skip-gram), 10 (cbow) |
| `min_count` | Min word frequency | 5+ (large corpus) |
| `sg` | Algorithm | 1=Skip-gram (thường tốt hơn) |
| `negative` | Negative samples | 5-20 |
| `epochs` | Training epochs | 5-10 (large corpus) |

---

## 10. Tóm tắt

- Word2Vec: học embeddings bằng prediction task
- 2 variants: **Skip-gram** (center→context) và **CBOW** (context→center)
- Output: **dense vectors** + **semantic properties**
- Gensim: thư viện Python phổ biến nhất để dùng Word2Vec
- Chương 3 tiếp theo: CBOW và Skip-gram chi tiết
