# 📙 Chương 3 - Bài 4: CBOW (Continuous Bag of Words)

---

## 1. CBOW là gì?

**CBOW** dự đoán **từ trung tâm** (target word) từ các **từ xung quanh** (context words).

```
Context words:     ["The", "___", "sat", "on"]
                         ↑
                    Target: "cat"

["The", "sat", "on"] → predict → "cat"
```

---

## 2. Architecture

```
Input Layer:
  one-hot(The) [V-dim]   ─┐
  one-hot(sat) [V-dim]   ─┤→ Average → [N-dim] → predict [V-dim]
  one-hot(on)  [V-dim]   ─┘   (embedding)         softmax

V = vocabulary size (~50,000)
N = embedding size (100-300)
```

---

## 3. Cách hoạt động từng bước

```python
import numpy as np

# Giả sử vocabulary nhỏ
vocab = ['The', 'cat', 'sat', 'on', 'mat', 'dog', 'ran', 'fast']
word2idx = {w: i for i, w in enumerate(vocab)}
V = len(vocab)
N = 3  # embedding dimension (nhỏ để demo)

# Corpus: "The cat sat on the mat"
corpus = ['The', 'cat', 'sat', 'on', 'mat']
window = 1

# Tạo training pairs (context → target)
def create_cbow_pairs(corpus, window):
    pairs = []
    for i, target in enumerate(corpus):
        context = []
        for j in range(max(0, i-window), min(len(corpus), i+window+1)):
            if i != j:
                context.append(corpus[j])
        if context:
            pairs.append((context, target))
    return pairs

pairs = create_cbow_pairs(corpus, window)
print("CBOW Training Pairs:")
for context, target in pairs:
    print(f"  Context: {context} → Target: {target}")
```

## 4. One-hot Encoding → Embedding

```python
def one_hot(word, vocab_dict):
    """Tạo one-hot vector"""
    v = np.zeros(len(vocab_dict))
    if word in vocab_dict:
        v[vocab_dict[word]] = 1
    return v

def get_context_vector(context_words, vocab_dict):
    """Average one-hot vectors của context words"""
    vectors = [one_hot(w, vocab_dict) for w in context_words]
    return np.mean(vectors, axis=0)

# Ví dụ
context = ['The', 'sat']
context_vec = get_context_vector(context, word2idx)
print(f"\nContext vector for {context}:")
print(context_vec)

# Nhân với weight matrix W → embedding

np.random.seed(42)
W_in = np.random.randn(V, N) * 0.01   # Input weights (V × N)
W_out = np.random.randn(N, V) * 0.01  # Output weights (N × V)

# Forward pass
hidden = context_vec @ W_in    # (1 × N)
print(f"\nHidden layer (embedding): {hidden}")

output = hidden @ W_out         # (1 × V)
def softmax(x):
    e_x = np.exp(x - x.max())
    return e_x / e_x.sum()
probs = softmax(output)
print(f"\nProbabilities (top 3):")
top3 = np.argsort(probs)[-3:][::-1]
for i in top3:
    print(f"  {vocab[i]}: {probs[i]:.4f}")
```

## 5. Training CBOW với Gensim

```python
from gensim.models import Word2Vec

# Corpus cho training
corpus = [
    "the cat sat on the mat".split(),
    "the dog ran on the grass".split(),
    "cats and dogs are common pets".split(),
    "the rat sat on the hat".split(),
    "I love my pet cat".split()
]

# CBOW: sg=0
cbow_model = Word2Vec(
    sentences=corpus,
    vector_size=50,
    window=2,
    min_count=1,
    sg=0,           # 0 = CBOW
    workers=2,
    epochs=200,
    seed=42
)

print(f"CBOW Vocabulary: {list(cbow_model.wv.key_to_index.keys())}")
print(f"\nVector for 'cat': {cbow_model.wv['cat'][:5]}...")

# Similarity
print("\nSimilarities:")
for pair in [('cat', 'dog'), ('cat', 'sat'), ('cat', 'mat')]:
    try:
        sim = cbow_model.wv.similarity(*pair)
        print(f"  {pair[0]} vs {pair[1]}: {sim:.4f}")
    except:
        pass
```

## 6. CBOW vs Skip-gram Comparison

| | CBOW | Skip-gram |
|-|------|-----------|
| Task | Context → Target | Target → Context |
| Training speed | Nhanh hơn | Chậm hơn |
| Quality (rare words) | Kém hơn | Tốt hơn |
| Quality (common words) | Tốt | Tốt |
| Memory | Ít hơn | Nhiều hơn |
| Dataset size | Large corpus tốt | Small to large |

**Quy tắc thực tế:** Skip-gram thường được ưa dùng hơn vì chất lượng cao hơn với rare words.

## 7. Các vấn đề với CBOW

### Hierarchical Softmax

Softmax trên 50,000 từ = **tốn kém**.

```
Thay vì: P(w|context) = softmax(V dimensions)
Hierarchical Softmax: dùng binary tree
→ Số phép tính: O(V) → O(log V)
```

### Negative Sampling

Thay vì compute full softmax, chỉ update:
- 1 positive sample (actual context word)
- k negative samples (random words)

```python
# Trong Gensim: negative sampling mặc định
model = Word2Vec(negative=5, hs=0)  # 5 negative samples
# hoặc
model = Word2Vec(hs=1)  # hierarchical softmax
```

---

## 8. Tóm tắt

- CBOW: **context → predict center**
- Nhanh hơn Skip-gram, tốt với phổ biến/frequent words
- Training: minimize cross-entropy loss giữa predicted và actual word
- Key trick: Negative Sampling hoặc Hierarchical Softmax để speed up
- Embedding = hàng của W_in matrix (sau khi train)
