# 📙 Chương 3 - Bài 12: Vấn đề OOV (Out-of-Vocabulary)

## 1. OOV là gì?

**OOV (Out-of-Vocabulary)**: từ xuất hiện trong test time nhưng không tồn tại trong vocabulary của model.

```
Training: "cat", "dog", "python" → VỚI trong vocab
Test: "doggy", "pythonic", "COVID-19" → KHÔNG có trong vocab → OOV
```

---

## 2. Tại sao OOV là vấn đề nghiêm trọng?

```python
from gensim.models import Word2Vec

corpus = [["cat", "dog", "python", "learning"]]
model = Word2Vec(corpus, vector_size=50, min_count=1, epochs=100, seed=42)

# Word có trong vocab → OK
print(model.wv["cat"])  # vector

# OOV → lỗi
try:
    vec = model.wv["doggy"]
except KeyError as e:
    print(f"OOV Error: {e}")  # "doggy" not in vocabulary

# Vấn đề thực tế:
oov_examples = [
    "COVID-19",        # từ mới xuất hiện
    "Googled",         # verb dạng morphological
    "unhappy",         # prefix
    "footballers",     # suffix plural
    "GPT-4",           # proper noun / model name
    "LeBron23",        # username
]
print("\nOOV words in real world:")
for w in oov_examples:
    in_vocab = w.lower() in model.wv
    print(f"  {w}: {'✅ in vocab' if in_vocab else '❌ OOV'}")
```

## 3. Các chiến lược xử lý OOV

### Strategy 1: <UNK> token
```python
import numpy as np
from typing import Optional

def get_embedding_unk(word: str, model, unk_strategy='zero'):
    """Xử lý OOV bằng UNK strategy"""
    if word in model.wv:
        return model.wv[word]
    
    if unk_strategy == 'zero':
        return np.zeros(model.vector_size)
    elif unk_strategy == 'random':
        return np.random.normal(0, 0.1, model.vector_size)
    elif unk_strategy == 'mean':
        # Mean của tất cả vectors
        all_vecs = model.wv.vectors
        return all_vecs.mean(axis=0)
    else:
        raise ValueError(f"Unknown strategy: {unk_strategy}")

# Demo
oov_word = "doggy"
print(f"OOV handlers for '{oov_word}':")
for strategy in ['zero', 'random', 'mean']:
    vec = get_embedding_unk(oov_word, model, strategy)
    print(f"  {strategy}: norm={np.linalg.norm(vec):.4f}")
```

### Strategy 2: Character-based partial matching
```python
def char_ngram_similarity(word: str, vocab_words: list, n: int = 3) -> str:
    """Tìm từ gần nhất bằng character n-gram"""
    def get_ngrams(w, n):
        w = f"_{w}_"
        return set(w[i:i+n] for i in range(len(w)-n+1))
    
    target_ngrams = get_ngrams(word.lower(), n)
    
    best_word, best_score = None, -1
    for vocab_word in vocab_words[:500]:  # limit for efficiency
        v_ngrams = get_ngrams(vocab_word, n)
        # Jaccard similarity
        inter = len(target_ngrams & v_ngrams)
        union = len(target_ngrams | v_ngrams)
        score = inter / union if union > 0 else 0
        if score > best_score:
            best_score = score
            best_word = vocab_word
    
    return best_word, best_score

# Demo
vocab = list(model.wv.key_to_index.keys())
oov_words = ["doggy", "cats", "pthon", "learnings"]
for w in oov_words:
    closest, score = char_ngram_similarity(w, vocab)
    print(f"  '{w}' → closest: '{closest}' (Jaccard={score:.3f})")
```

### Strategy 3: Morphological decomposition
```python
def morphological_fallback(word: str, model) -> Optional[np.ndarray]:
    """Strip affixes to find root in vocab"""
    word = word.lower()
    
    if word in model.wv:
        return model.wv[word]
    
    # Suffix stripping
    suffixes = ['ing', 'ings', 'ed', 'er', 'ers', 'est',
                'ly', 'ment', 'ness', 'ful', 'less', 'tion',
                's', 'es', 'ies']
    for suffix in sorted(suffixes, key=len, reverse=True):
        if word.endswith(suffix) and len(word) - len(suffix) >= 3:
            root = word[:-len(suffix)]
            if root in model.wv:
                return model.wv[root]
    
    # Prefix stripping
    prefixes = ['un', 're', 'dis', 'mis', 'pre', 'over', 'under']
    for prefix in prefixes:
        if word.startswith(prefix) and len(word) - len(prefix) >= 3:
            root = word[len(prefix):]
            if root in model.wv:
                return model.wv[root]
    
    return None  # Still OOV

# Demo
oov_tests = ["unhappy", "runners", "relearning", "doggy"]
for w in oov_tests:
    vec = morphological_fallback(w, model)
    if vec is not None:
        print(f"  '{w}' → handled via morphology ✅ (norm={np.linalg.norm(vec):.3f})")
    else:
        print(f"  '{w}' → still OOV ❌")
```

## 4. So sánh chiến lược OOV

| Chiến lược | Đơn giản | Hiệu quả | FastText thay thế |
|-----------|---------|---------|------------------|
| <UNK> zero | ✅ | ❌ | — |
| <UNK> mean | ✅ | Trung bình | — |
| Char n-gram | Trung bình | Tốt | ✅ |
| Morphological | Phức tạp | Tốt | ✅ |
| **FastText** | ✅ (built-in) | **Rất tốt** | N/A |

## 5. Thực tế: FastText giải quyết OOV natively

```python
from gensim.models import FastText

ft_model = FastText(corpus, vector_size=50, min_count=1,
                    min_n=3, max_n=6, epochs=100, seed=42)

# FastText xử lý OOV qua subword
oov_tests = ["doggy", "cats", "pythons", "learner"]
for w in oov_tests:
    vec = ft_model.wv[w]  # Không bao giờ KeyError!
    print(f"  '{w}': norm={np.linalg.norm(vec):.3f} ✅")
```

## 6. Tóm tắt

- OOV là vấn đề phổ biến với Word2Vec/GloVe
- Chiến lược đơn giản: dùng zero/mean vector
- Chiến lược tốt: morphological fallback, char n-gram
- **Giải pháp tốt nhất**: dùng **FastText** (subword) hoặc **BPE tokenizer** (BERT/GPT era)
