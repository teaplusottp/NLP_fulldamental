## 1. Transfer Learning là gì?

**Transfer Learning**: tận dụng kiến thức học từ task/domain A để giải task/domain B.

```
Tương tự con người:
  Biết tiếng Anh → học tiếng Đức dễ hơn
  Biết cờ vua → học cờ tướng nhanh hơn
  
Trong NLP:
  Pretrained LM (trên 100GB text) → Fine-tune cho Sentiment Analysis
```

---

## 2. Hai paradigms Transfer Learning trong NLP

```
Paradigm 1 (Pre-2018): Feature-based
  Pretrained embed (W2V/GloVe) → cố định
  → Task model học trên top of features
  
Paradigm 2 (2018+): Fine-tuning
  Pretrained LM (ELMo/BERT/GPT) → tiếp tục train
  → Toàn bộ model adapted
```

## 3. Feature-based Transfer Learning

```python
import numpy as np
from sklearn.linear_model import LogisticRegression
from sklearn.model_selection import cross_val_score
from sklearn.preprocessing import LabelEncoder
from gensim.models import Word2Vec
import re

# Pretrained base model (simulate với Word2Vec)
pretrain_corpus = [
    ["the", "cat", "is", "cute", "and", "fluffy"],
    ["machine", "learning", "uses", "data", "to", "learn"],
    ["this", "restaurant", "is", "amazing", "and", "delicious"],
    ["terrible", "service", "horrible", "experience", "bad"],
    ["neural", "network", "deep", "learning", "model"],
    ["natural", "language", "processing", "text", "analysis"],
    ["happy", "joyful", "wonderful", "great", "excellent"],
    ["sad", "depressed", "awful", "negative", "terrible"],
] * 30

pretrained_wv = Word2Vec(
    pretrain_corpus, vector_size=100, window=5,
    min_count=1, sg=1, epochs=200, seed=42
)
print(f"Pretrained vocab: {len(pretrained_wv.wv)}")

# -------------------------
# Task: Sentiment Analysis
# -------------------------
labeled_data = [
    ("the service is great and food is amazing", 1),
    ("horrible experience terrible service", 0),
    ("wonderful product excellent quality", 1),
    ("very bad product awful experience", 0),
    ("amazing restaurant delicious food happy", 1),
    ("terrible product bad quality", 0),
    ("excellent great wonderful", 1),
    ("sad terrible awful negative bad", 0),
]

def mean_embed(text: str, wv) -> np.ndarray:
    tokens = re.findall(r'[a-z]+', text.lower())
    vecs = [wv[w] for w in tokens if w in wv]
    return np.mean(vecs, axis=0) if vecs else np.zeros(100)

# Feature-based: use pretrained vectors as fixed features
X_feat = np.array([mean_embed(t, pretrained_wv.wv) for t, _ in labeled_data])
y = np.array([label for _, label in labeled_data])

clf_feat = LogisticRegression(max_iter=300, random_state=42)
scores_feat = cross_val_score(clf_feat, X_feat, y, cv=min(4, len(set(y))*2))
print(f"\nFeature-based accuracy: {scores_feat.mean():.2%} ± {scores_feat.std():.2%}")
```

## 4. Fine-tuning Paradigm

```python
# Fine-tuning: continue training model on task data
import copy

task_corpus = [
    ["great", "amazing", "wonderful", "delicious", "excellent"],
    ["terrible", "horrible", "awful", "bad", "disgusting"],
    ["service", "food", "restaurant", "experience", "quality"],
] * 20

# Clone pretrained model
finetuned_wv = copy.deepcopy(pretrained_wv)

# Tạo TaskTaggedCorpus
task_sentences = [re.findall(r'[a-z]+', t.lower()) for t, _ in labeled_data]
valid_sentences = [s for s in task_sentences if s]

# Build vocab và fine-tune
finetuned_wv.build_vocab(valid_sentences, update=True)
finetuned_wv.train(
    valid_sentences,
    total_examples=len(valid_sentences),
    epochs=100
)

X_ft = np.array([mean_embed(t, finetuned_wv.wv) for t, _ in labeled_data])
clf_ft = LogisticRegression(max_iter=300, random_state=42)
scores_ft = cross_val_score(clf_ft, X_ft, y, cv=min(4, len(set(y))*2))
print(f"Fine-tuned accuracy: {scores_ft.mean():.2%} ± {scores_ft.std():.2%}")
```

## 6. ULMFiT: 3-phase Fine-tuning

```python
print("""
ULMFiT (Howard & Ruder 2018) - pioneered LM fine-tuning:

Phase 1: LM pre-training
  Train 3-layer LSTM on Wikipedia → learn general language

Phase 2: Domain LM fine-tuning
  Continue training on target domain text (no labels needed)
  → slanted triangular LR schedule
  → discriminative fine-tuning (different LR per layer)

Phase 3: Task fine-tuning
  Add classification head → fine-tune on labeled task data
  → gradual unfreezing: unfreeze 1 layer per epoch
  
Key innovations:
  1. Slanted triangular LR: sharp rise → linear decay
  2. Discriminative LR: lower LR for early layers
  3. Gradual unfreezing: prevent catastrophic forgetting
""")
```

## 7. Tóm tắt Transfer Learning

| Method | Year | Paradigm | Best For |
|--------|------|---------|---------|
| W2V + classifier | 2013 | Feature-based | Simple tasks |
| ELMo + task | 2018 | Feature-based | Good baseline |
| ULMFiT | 2018 | Fine-tuning LM | Text classification |
| BERT fine-tune | 2018 | Fine-tuning | Most NLU tasks |
| GPT fine-tune | 2018 | Fine-tuning | Generation |
| LoRA/PEFT | 2022 | Parameter-efficient | Large models |

- Transfer learning là nền tảng của NLP hiện đại
- BERT → fine-tune → trở thành tiêu chuẩn cho classification, NER, QA
- Chương 5 sẽ cover BERT, GPT chi tiết
