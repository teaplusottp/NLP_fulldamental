---

## 1. GloVe là gì?

**GloVe (Global Vectors for Word Representation)** - Pennington et al., 2014 (Stanford).

**Key idea:** Kết hợp **global statistics** (co-occurrence matrix của toàn corpus) với **local context window** (như Word2Vec).

---

## 2. Motivation

### Word2Vec:
- Local context windows
- Không tận dụng global co-occurrence statistics

### GloVe:
- Dùng **toàn bộ co-occurrence matrix** của corpus
- Embeddings phản ánh **tỉ lệ probability** co-occurrence

---

## 3. Ý tưởng cốt lõi

Xét ratio: $P(k|ice) / P(k|steam)$

| Word k | P(k\|ice) | P(k\|steam) | Ratio |
|--------|-----------|-------------|-------|
| solid | high | low | **large** (>1) |
| gas | low | high | **small** (<1) |
| water | high | high | ≈ 1 |
| fashion | low | low | ≈ 1 |

**Ratio** phân biệt tốt hơn raw probability!

GloVe objective: $\vec{w_i} \cdot \vec{w_j} \approx \log P(j|i)$

---

## 4. GloVe Loss Function

$$J = \sum_{i,j=1}^{V} f(X_{ij}) (w_i^T \tilde{w}_j + b_i + \tilde{b}_j - \log X_{ij})^2$$

- $X_{ij}$: co-occurrence count of words i and j
- $f(X_{ij})$: weighting function (giảm ảnh hưởng của very rare và very common pairs)
- $w_i$: word vector, $\tilde{w}_j$: context vector

import numpy as np

def load_glove_vectors(filepath, max_words=None):
    """Load GloVe vectors từ file"""
    embeddings = {}
    with open(filepath, 'r', encoding='utf-8') as f:
        for i, line in enumerate(f):
            if max_words and i >= max_words:
                break
            parts = line.strip().split()
            word = parts[0]
            vector = np.array(parts[1:], dtype=np.float32)
            embeddings[word] = vector
    return embeddings

## 6. Similarity với GloVe

```python
from sklearn.metrics.pairwise import cosine_similarity

def glove_similarity(word1, word2, embeddings):
    if word1 not in embeddings or word2 not in embeddings:
        return None
    v1 = embeddings[word1].reshape(1, -1)
    v2 = embeddings[word2].reshape(1, -1)
    return cosine_similarity(v1, v2)[0][0]

def most_similar_glove(word, embeddings, topn=5):
    if word not in embeddings:
        return []
    query = embeddings[word]
    scores = {}
    for w, vec in embeddings.items():
        if w != word:
            sim = cosine_similarity(query.reshape(1,-1), vec.reshape(1,-1))[0][0]
            scores[w] = sim
    return sorted(scores.items(), key=lambda x: x[1], reverse=True)[:topn]

print("\nGloVe Similarities (mock):")
pairs = [('king', 'queen'), ('good', 'great'), ('cat', 'dog'), ('good', 'bad')]
for w1, w2 in pairs:
    sim = glove_similarity(w1, w2, mock_glove)
    if sim:
        print(f"  {w1} vs {w2}: {sim:.4f}")

print("\nMost similar to 'good':")
for word, score in most_similar_glove('good', mock_glove):
    print(f"  {word}: {score:.4f}")
```

## 7. GloVe cho NLP Tasks

```python
def get_sentence_embedding_glove(sentence, embeddings, method='mean'):
    """Sentence embedding từ GloVe word vectors"""
    words = sentence.lower().split()
    vectors = [embeddings[w] for w in words if w in embeddings]
    
    if not vectors:
        return np.zeros(100)  # fallback
    
    if method == 'mean':
        return np.mean(vectors, axis=0)
    elif method == 'max':
        return np.max(vectors, axis=0)
    elif method == 'sum':
        return np.sum(vectors, axis=0)

# Sentence similarity
sentences = [
    "the king ruled the kingdom",
    "the queen led the empire",
    "the cat sat on the mat",
    "the dog ran in the park"
]

embeddings_sentences = [get_sentence_embedding_glove(s, mock_glove) for s in sentences]

print("\nSentence Similarities (GloVe mean pooling):")
for i in range(len(sentences)):
    for j in range(i+1, len(sentences)):
        v1 = embeddings_sentences[i].reshape(1,-1)
        v2 = embeddings_sentences[j].reshape(1,-1)
        sim = cosine_similarity(v1, v2)[0][0]
        print(f"  '{sentences[i][:20]}' vs '{sentences[j][:20]}': {sim:.4f}")
```

## 9. So sánh Word2Vec vs GloVe

| | Word2Vec | GloVe |
|-|----------|-------|
| Approach | Local context window | Global co-occurrence |
| Training | SGD, online | Weighted least squares |
| Analogy tasks | Tốt | Tốt |
| Rare words | Kém hơn | Tốt hơn |
| Memory | Compact | Cần store co-occ matrix |
| Available pretrained | Có | Có (6B/42B/840B tokens) |

**Thực tế:** Cả hai tương đương; GloVe đôi khi tốt hơn với analogy tasks.

---

## 10. Tóm tắt

- GloVe = **global statistics** + **local context**
- Dùng tỉ lệ co-occurrence probability, không phải raw count
- Available pretrained: 50d, 100d, 200d, 300d
- Sentence embedding: mean/max pooling qua word vectors
- Thực tế: GloVe và Word2Vec có performance tương tự
