## 1. Vấn đề Curse of Dimensionality

TF-IDF vector: 50,000 dimensions cho 50,000 từ trong vocabulary.

- Hầu hết values = 0 (rất sparse)
- Khó visualize
- KNN/similarity tệ trong high-dimensional space

→ **Dimensionality Reduction**: giảm số chiều xuống còn k chiều.

---

## 2. PCA (Principal Component Analysis)

PCA tìm hướng (components) chứa **nhiều variance nhất** trong data, rồi project data vào k hướng đó.

```python
from sklearn.decomposition import PCA, TruncatedSVD
from sklearn.feature_extraction.text import TfidfVectorizer
import numpy as np
import matplotlib.pyplot as plt

texts = [
    "football match score goal penalty",
    "basketball slam dunk championship",
    "tennis racket court wimbledon",
    "python code programming algorithm",
    "machine learning deep neural network",
    "AI data science model training",
    "election president vote parliament",
    "government law policy economy",
    "music concert singer album tour",
    "movie film director actor cinema"
]
labels_str = ['Sports']*3 + ['Tech']*3 + ['Politics']*2 + ['Entertainment']*2

vec = TfidfVectorizer()
X = vec.fit_transform(texts)

# PCA không hoạt động trực tiếp với sparse matrices
# → Dùng TruncatedSVD (Latent Semantic Analysis)
```

---

## 3. LSA / TruncatedSVD (tốt hơn PCA cho text)

```python
from sklearn.decomposition import TruncatedSVD

# Giảm từ 50000 dims xuống 50 dims
svd = TruncatedSVD(n_components=50, random_state=42)
X_reduced = svd.fit_transform(X)

print(f"Original shape: {X.shape}")
print(f"Reduced shape: {X_reduced.shape}")
print(f"Explained variance: {svd.explained_variance_ratio_.sum():.2%}")
```

**LSA (Latent Semantic Analysis)** = TF-IDF + TruncatedSVD

Giúp capture **semantic similarity**: "car" và "automobile" sẽ gần nhau trong reduced space.

## 5. PCA cho Dense Features

```python
from sklearn.decomposition import PCA
import numpy as np

# Với dense features (sau khi toarray())
X_dense = X.toarray()

pca = PCA(n_components=10, random_state=42)
X_pca = pca.fit_transform(X_dense)

# Explained variance per component
print("Explained variance ratio:")
for i, ev in enumerate(pca.explained_variance_ratio_):
    print(f"  PC{i+1}: {ev:.4f} ({ev*100:.1f}%)")
    
print(f"\nTotal: {pca.explained_variance_ratio_.sum():.4f}")

# Chọn n_components giải thích 95% variance
pca_full = PCA(n_components=0.95, random_state=42)  # 95% variance
X_pca95 = pca_full.fit_transform(X_dense)
print(f"\n95% variance needs: {pca_full.n_components_} components")
```

## 6. LSA trong Pipeline cho Classification

```python
from sklearn.pipeline import Pipeline
from sklearn.linear_model import LogisticRegression
from sklearn.model_selection import cross_val_score

# No reduction
pipeline_baseline = Pipeline([
    ('tfidf', TfidfVectorizer()),
    ('clf', LogisticRegression(max_iter=1000))
])

# With LSA
pipeline_lsa = Pipeline([
    ('tfidf', TfidfVectorizer()),
    ('lsa', TruncatedSVD(n_components=50, random_state=42)),
    ('clf', LogisticRegression(max_iter=1000))
])

for name, pipe in [('Baseline', pipeline_baseline), ('LSA-50', pipeline_lsa)]:
    scores = cross_val_score(pipe, texts, labels_str, cv=3, scoring='accuracy')
    print(f"{name}: {scores.mean():.4f} ± {scores.std():.4f}")
```

## 8. Chọn n_components

```python
# Scree plot: tìm "elbow"
svd_full = TruncatedSVD(n_components=min(X.shape)-1, random_state=42)
svd_full.fit(X)

cumsum = np.cumsum(svd_full.explained_variance_ratio_)
n_95 = np.argmax(cumsum >= 0.95) + 1
n_99 = np.argmax(cumsum >= 0.99) + 1

print(f"Components for 95% variance: {n_95}")
print(f"Components for 99% variance: {n_99}")
```

## 9. Tóm tắt

| Method | Sparse | Semantic | Neural |
|--------|--------|----------|--------|
| Feature Selection | ✅ Giữ sparse | ❌ | Không |
| LSA/TruncatedSVD | ✅ Tốt nhất | ✅ | Không |
| PCA | ❌ Cần dense | ✅ | Không |
| t-SNE | ❌ Cần dense | ✅ | Chỉ visualize |

- **LSA = TF-IDF + TruncatedSVD**: chuẩn cho text
- Thường dùng 100-300 components
- Giúp capture semantic similarity tốt hơn raw TF-IDF
