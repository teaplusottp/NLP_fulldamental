## 1. Decision Tree là gì?

**Decision Tree** học một cây quyết định dựa trên câu hỏi Yes/No về features.

```
            [has "free"?]
            /           \
          Yes            No
    [has "win"?]    [has "meeting"?]
    /       \           /        \
  Yes        No       Yes         No
 SPAM     NOT SPAM  NOT SPAM    NOT SPAM
```

---

## 2. Cơ chế hoạt động

### Splitting Criteria

Tại mỗi node, chọn feature split dựa trên:

**Gini Impurity:**
$$Gini = 1 - \sum_{i} p_i^2$$

**Information Gain (Entropy):**
$$H = -\sum_{i} p_i \log_2(p_i)$$

→ Chọn split nào giảm impurity nhiều nhất.

## 3. Code Python

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.tree import DecisionTreeClassifier, export_text
from sklearn.model_selection import train_test_split
from sklearn.metrics import classification_report

texts = [
    "free money win prize now",
    "click here win lottery",
    "meeting tomorrow 3pm",
    "project review deadline",
    "buy cheap medicine",
    "urgent limited offer",
    "team lunch wednesday",
    "quarterly report due"
]
labels = [1, 1, 0, 0, 1, 1, 0, 0]  # 1=spam

X_train, X_test, y_train, y_test = train_test_split(
    texts, labels, test_size=0.25, random_state=42
)

vec = TfidfVectorizer()
X_train_vec = vec.fit_transform(X_train)
X_test_vec = vec.transform(X_test)

dt = DecisionTreeClassifier(
    criterion='gini',       # 'gini' hoặc 'entropy'
    max_depth=5,            # giới hạn depth để tránh overfit
    min_samples_split=2,    # min samples để split
    min_samples_leaf=1,     # min samples tại leaf
    random_state=42
)
dt.fit(X_train_vec, y_train)

y_pred = dt.predict(X_test_vec)
print(classification_report(y_test, y_pred))
```

## 6. Vấn đề Overfitting

Decision Tree **dễ overfit** mạnh nếu không giới hạn:

```
max_depth=None → cây học thuộc lòng training data
→ accuracy train: 100%
→ accuracy test: thấp
```

**Cách giảm overfitting:**

```python
# Kiểm soát độ phức tạp cây
dt = DecisionTreeClassifier(
    max_depth=10,           # giới hạn depth
    min_samples_split=5,    # ít nhất 5 samples mới split
    min_samples_leaf=2,     # ít nhất 2 samples mỗi leaf
    max_features=0.8        # chỉ dùng 80% features mỗi split
)
```

## 7. So sánh Decision Tree vs Linear Models

| | Decision Tree | Logistic Reg. / SVM |

| Interpretable | Rất cao | Trung bình |
