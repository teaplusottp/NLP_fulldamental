# 📗 Chương 2 - Bài 16: Feature Selection

---

## 1. Feature Selection là gì?

**Feature Selection** = chọn **subset features quan trọng nhất** từ feature space, loại bỏ noise và redundant features.

**Lợi ích:**
- Giảm overfitting
- Train nhanh hơn
- Model đơn giản hơn, dễ interpret
- Đôi khi tăng accuracy

---

## 2. Tại sao quan trọng trong NLP?

TF-IDF với vocabulary = 50,000 từ → 50,000 features!

- Nhiều từ không liên quan (noise)
- Sparse vectors cực kỳ thưa
- Tính toán chậm

→ Feature selection giúp giữ chỉ top k features quan trọng.

---

## 3. Phương pháp 1: Chi-Square Test (SelectKBest)

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.feature_selection import SelectKBest, chi2, mutual_info_classif
from sklearn.linear_model import LogisticRegression
from sklearn.pipeline import Pipeline
import numpy as np

texts = [
    "sports football basketball score",
    "tennis match championship court",
    "python programming code algorithm",
    "machine learning neural network",
    "election vote president government",
    "parliament law policy reform",
    "music concert album singer",
    "movie film actor director"
]
labels = [0,0,1,1,2,2,3,3]  # 4 classes

vec = TfidfVectorizer()
X = vec.fit_transform(texts)

# Chi-square: chọn top 5 features
selector = SelectKBest(chi2, k=5)
X_selected = selector.fit_transform(X, labels)

# Xem features được chọn
feature_names = vec.get_feature_names_out()
selected_mask = selector.get_support()
selected_features = feature_names[selected_mask]
print("Selected features:", selected_features)
print("Original shape:", X.shape, "→ Selected shape:", X_selected.shape)
```

## 4. Phương pháp 2: Mutual Information

```python
# Mutual Information: đo lường dependency giữa feature và label
selector_mi = SelectKBest(mutual_info_classif, k=10)
X_mi = selector_mi.fit_transform(X, labels)

# Scores mỗi feature
mi_scores = mutual_info_classif(X, labels)
feature_names = vec.get_feature_names_out()

# Sort theo score
sorted_idx = np.argsort(mi_scores)[::-1]
print("\nTop 10 features theo Mutual Information:")
for i in sorted_idx[:10]:
    if mi_scores[i] > 0:
        print(f"  '{feature_names[i]}': {mi_scores[i]:.4f}")
```

---

## 5. Phương pháp 3: Variance Threshold

Loại bỏ features có variance thấp (không thay đổi nhiều):

```python
from sklearn.feature_selection import VarianceThreshold

# Loại bỏ features xuất hiện trong < 20% documents
selector_var = VarianceThreshold(threshold=0.01)
X_var = selector_var.fit_transform(X.toarray())
print(f"\nAfter VarianceThreshold: {X.shape} → {X_var.shape}")
```

---

## 6. Phương pháp 4: Model-based Selection

Dùng model weights để chọn features:

```python
from sklearn.feature_selection import SelectFromModel
from sklearn.svm import LinearSVC

# SVM với L1 penalty → sparse weights
l1_svm = LinearSVC(C=0.01, penalty='l1', dual=False, max_iter=2000)
selector_model = SelectFromModel(l1_svm)
selector_model.fit(X, labels)
X_l1 = selector_model.transform(X)
print(f"\nAfter L1 SVM selection: {X.shape} → {X_l1.shape}")

# Logistic Regression với L1
from sklearn.linear_model import LogisticRegression
l1_lr = LogisticRegression(penalty='l1', C=0.1, solver='saga', max_iter=1000)
selector_lr = SelectFromModel(l1_lr)
selector_lr.fit(X, labels)
X_lr = selector_lr.transform(X)
print(f"After L1 LR selection: {X.shape} → {X_lr.shape}")
```

---

## 7. Phương pháp 5: RFE (Recursive Feature Elimination)

```python
from sklearn.feature_selection import RFE

lr = LogisticRegression(max_iter=1000, random_state=42)
rfe = RFE(estimator=lr, n_features_to_select=10, step=5)
rfe.fit(X, labels)

selected = feature_names[rfe.support_]
print("\nRFE selected features:", selected)
```

## 8. Pipeline với Feature Selection

```python
from sklearn.pipeline import Pipeline
from sklearn.model_selection import cross_val_score

# Compare: no selection vs chi2 selection
pipeline_no_sel = Pipeline([
    ('tfidf', TfidfVectorizer()),
    ('clf', LogisticRegression(max_iter=1000, random_state=42))
])

pipeline_chi2 = Pipeline([
    ('tfidf', TfidfVectorizer()),
    ('select', SelectKBest(chi2, k=20)),
    ('clf', LogisticRegression(max_iter=1000, random_state=42))
])

for name, pipe in [('No Selection', pipeline_no_sel),
                   ('Chi2 k=20', pipeline_chi2)]:
    scores = cross_val_score(pipe, texts, labels, cv=3, scoring='accuracy')
    print(f"{name}: {scores.mean():.4f} ± {scores.std():.4f}")
```

## 9. So sánh Phương pháp

| Phương pháp | Speed | Dùng khi |
|-------------|-------|----------|
| Chi-Square | Nhanh | Categorical features, text |
| Mutual Info | Trung bình | Non-linear relationships |
| Variance Threshold | Rất nhanh | Loại bỏ zero-variance |
| L1 Model | Chậm | Feature selection + model |
| RFE | Rất chậm | Dataset nhỏ, interpretability |

---

## 10. Tóm tắt

- Feature selection giảm dimensionality, tránh overfitting
- **Chi-Square + SelectKBest**: chuẩn nhất cho text classification
- L1 (Lasso) regression: tự động feature selection
- Luôn compare với/không có feature selection qua CV
