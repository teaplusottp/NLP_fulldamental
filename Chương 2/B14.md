## 1. Binary vs Multi-class vs Multi-label

| Type | Ví dụ | Labels per sample |
|------|-------|-------------------|
| **Binary** | Spam/Ham | 1 label, 2 classes |
| **Multi-class** | Tin tức: Sports/Tech/Politics | 1 label, n > 2 classes |
| **Multi-label** | Bài viết có thể: Sports + Health | Nhiều labels cùng lúc |

---

## 2. Multi-class Classification

### Các chiến lược:

**One-vs-Rest (OvR):**
- Train k binary classifiers (k = số class)
- Classifier i: "class i vs all other classes"
- Predict: chọn class có score cao nhất

**One-vs-One (OvO):**
- Train C(k,2) = k*(k-1)/2 classifiers
- Mỗi classifier phân biệt 2 class
- Predict: voting

---

## 3. Code Multi-class

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.linear_model import LogisticRegression
from sklearn.multiclass import OneVsRestClassifier, OneVsOneClassifier
from sklearn.metrics import classification_report
from sklearn.model_selection import train_test_split

# News dataset: 4 categories
texts = [
    "football match goals scored penalty",
    "basketball NBA championship finals",
    "tennis grand slam wimbledon",
    "python machine learning algorithm",
    "deep learning neural network GPU",
    "database SQL programming code",
    "election president vote government",
    "parliament law policy debate",
    "economy tax budget financial",
    "music concert album release singer",
    "movie film actor director award",
    "theater performance dance art"
]
# 0=Sports, 1=Tech, 2=Politics, 3=Entertainment
labels = [0,0,0,1,1,1,2,2,2,3,3,3]

X_train, X_test, y_train, y_test = train_test_split(
    texts, labels, test_size=0.25, random_state=42, stratify=labels
)

vec = TfidfVectorizer(ngram_range=(1,2))
X_train_vec = vec.fit_transform(X_train)
X_test_vec = vec.transform(X_test)

# LogisticRegression tự động dùng OvR
lr = LogisticRegression(
    multi_class='ovr',      # hoặc 'multinomial'
    max_iter=1000,
    random_state=42
)
lr.fit(X_train_vec, y_train)
y_pred = lr.predict(X_test_vec)

print(classification_report(
    y_test, y_pred,
    target_names=['Sports', 'Tech', 'Politics', 'Entertainment']
))
```

---

## 4. Softmax (Multinomial Logistic Regression)

Thay vì sigmoid, dùng softmax cho multi-class:

$$P(y=k|x) = \frac{e^{w_k^T x}}{\sum_{j=1}^{K} e^{w_j^T x}}$$

```python
# Multinomial = dùng softmax
lr_multi = LogisticRegression(
    multi_class='multinomial',
    solver='lbfgs',
    max_iter=1000
)
lr_multi.fit(X_train_vec, y_train)
proba = lr_multi.predict_proba(X_test_vec)
print("Class probabilities:", proba[:2].round(3))
```

## 5. Multi-label Classification

Mỗi sample có thể thuộc **nhiều class** cùng lúc:

```python
from sklearn.multiclass import OneVsRestClassifier
from sklearn.preprocessing import MultiLabelBinarizer
import numpy as np

# Bài viết có thể thuộc nhiều chủ đề
texts_ml = [
    "sports health running marathon fitness",
    "tech AI health digital medicine",
    "sports tech esport gaming tournament",
    "health medicine diet nutrition",
    "tech programming software developer",
    "sports running fitness exercise health"
]

# Multi-label: mỗi sample có nhiều labels
labels_ml = [
    ['sports', 'health'],
    ['tech', 'health'],
    ['sports', 'tech'],
    ['health'],
    ['tech'],
    ['sports', 'health']
]

# Binarize labels
mlb = MultiLabelBinarizer()
Y = mlb.fit_transform(labels_ml)
print("Classes:", mlb.classes_)
print("Label matrix:\n", Y)
```

---

## 6. Train Multi-label Model

```python
# Train/test split cho multi-label
X_ml = vec.fit_transform(texts_ml)
X_tr, X_te, Y_tr, Y_te = train_test_split(X_ml, Y, test_size=0.3, random_state=42)

# OneVsRestClassifier: train k binary classifiers
ovr = OneVsRestClassifier(LogisticRegression(max_iter=1000))
ovr.fit(X_tr, Y_tr)

Y_pred = ovr.predict(X_te)
print("Predicted labels:")
for i, pred in enumerate(Y_pred):
    print(f"  {mlb.inverse_transform(pred.reshape(1,-1))}")
```

---

## 7. Metrics cho Multi-label

```python
from sklearn.metrics import (
    hamming_loss, jaccard_score, f1_score
)

# Hamming Loss: fraction of wrong labels
hl = hamming_loss(Y_te, Y_pred)
print(f"Hamming Loss: {hl:.4f}")  # Nhỏ hơn = tốt hơn

# F1 (micro, macro, weighted)
f1_micro = f1_score(Y_te, Y_pred, average='micro')
f1_macro = f1_score(Y_te, Y_pred, average='macro')
print(f"F1 Micro: {f1_micro:.4f}")
print(f"F1 Macro: {f1_macro:.4f}")
```

## 8. Tóm tắt

| | Multi-class | Multi-label |
|-|-------------|-------------|
| Labels/sample | 1 | ≥ 1 |
| Ví dụ | Phân loại tin tức | Tag bài viết |
| Chiến lược | OvR, Multinomial | OvR, Binary Relevance |
| Metric | Accuracy, F1 | Hamming Loss, F1 |

- sklearn LogisticRegression tự xử lý multi-class
- Multi-label: dùng `OneVsRestClassifier` + `MultiLabelBinarizer`
