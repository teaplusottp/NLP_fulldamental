## 2. Bag of Words (BoW)

### Ý tưởng:
- Tạo một **từ điển (vocabulary)** từ tất cả các từ
- Mỗi câu = vector: **số lần xuất hiện** của từng từ trong từ điển

### Ví dụ:

**Corpus:**
```
Doc1: "tôi thích học AI"
Doc2: "tôi thích ăn cơm"
Doc3: "AI rất hay"
```

**Vocabulary:** [tôi, thích, học, AI, ăn, cơm, rất, hay]

**Vector:**
```
Doc1: [1, 1, 1, 1, 0, 0, 0, 0]
Doc2: [1, 1, 0, 0, 1, 1, 0, 0]
Doc3: [0, 0, 0, 1, 0, 0, 1, 1]
```

---

## 3. TF-IDF (ôn lại từ Chương 1)

**TF-IDF = TF × IDF**

| Thành phần | Ý nghĩa |
|------------|---------|
| TF | Từ xuất hiện nhiều trong document |
| IDF | Từ hiếm trong toàn bộ corpus |
| TF-IDF | Từ quan trọng trong document nhưng hiếm globally |

---

## 4. Code Python

### BoW với sklearn:

```python
from sklearn.feature_extraction.text import CountVectorizer

corpus = [
    "tôi thích học AI",
    "tôi thích ăn cơm",
    "AI rất hay"
]

vectorizer = CountVectorizer()
X = vectorizer.fit_transform(corpus)

print("Vocabulary:", vectorizer.vocabulary_)
print("BoW matrix:\n", X.toarray())
```

### TF-IDF với sklearn:

```python
from sklearn.feature_extraction.text import TfidfVectorizer

vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(corpus)

print("TF-IDF matrix:\n", X.toarray())
print("Feature names:", vectorizer.get_feature_names_out())
```

---

## 5. Nhược điểm BoW

❌ Không giữ thứ tự từ: "dog bites man" ≠ "man bites dog" nhưng BoW giống nhau  
❌ Vector cực thưa (sparse) khi vocabulary lớn  
❌ Không capture ngữ nghĩa: "good" và "great" hoàn toàn độc lập  

---

## 6. N-gram BoW

Giải quyết vấn đề thứ tự từ bằng cách dùng bigram, trigram:

```python
vectorizer = CountVectorizer(ngram_range=(1, 2))  # unigram + bigram
X = vectorizer.fit_transform(corpus)
print(vectorizer.get_feature_names_out())
```

---

## 7. So sánh BoW vs TF-IDF

| | BoW | TF-IDF |
|-|-----|--------|
| Trọng số | Tần suất thô | Tần suất có trọng số IDF |
| Ưu điểm | Đơn giản | Giảm ảnh hưởng stopwords |
| Nhược điểm | Không phân biệt quan trọng | Phức tạp hơn |
| Dùng khi | Baseline | Hầu hết NLP tasks |

---

## 8. Thực hành: Phân loại đơn giản

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.naive_bayes import MultinomialNB

# Dữ liệu
train_texts = ["spam free money", "win prize now", "hello friend", "project meeting"]
train_labels = [1, 1, 0, 0]  # 1=spam, 0=not spam

# Vector hóa
vec = TfidfVectorizer()
X_train = vec.fit_transform(train_texts)

# Train
clf = MultinomialNB()
clf.fit(X_train, train_labels)

# Test
test = vec.transform(["free money win now"])
print(clf.predict(test))  # → [1]
```
