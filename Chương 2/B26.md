## 1. Overview

| | Classical ML | Deep Learning |
|-|-------------|---------------|
| **Features** | Thủ công (TF-IDF, BoW) | Tự học từ data |
| **Data** | Ít data vẫn tốt | Cần nhiều data |
| **Training** | Nhanh (seconds - minutes) | Chậm (hours - days) |
| **Inference** | Rất nhanh | Tương đối nhanh |
| **Interpretability** | Cao | Thấp |
| **Setup** | Dễ | Phức tạp hơn |
| **Performance (small data)** | ⭐⭐⭐⭐ | ⭐⭐⭐ |
| **Performance (large data)** | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |

---

## 2. Khi nào dùng Classical ML?

✅ **Dùng ML khi:**
- Dataset nhỏ (< 10,000 samples)
- Cần train/deploy nhanh
- Cần interpretability (giải thích được)
- Tài nguyên tính toán hạn chế
- Baseline cần nhanh
- Simple classification (spam, sentiment đơn giản)

---

## 3. Khi nào dùng Deep Learning?

✅ **Dùng DL khi:**
- Dataset lớn (> 100,000 samples)
- Hiểu ngữ nghĩa sâu (sarcasm, implicit meaning)
- NLU phức tạp (question answering, NER)
- Cần SOTA performance
- Có GPU và thời gian

## 4. Benchmark Thực tế

```python
import time
import numpy as np
from sklearn.pipeline import Pipeline
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.linear_model import LogisticRegression
from sklearn.svm import LinearSVC
from sklearn.metrics import f1_score
from sklearn.model_selection import train_test_split
import warnings
warnings.filterwarnings('ignore')

# Tạo synthetic dataset lớn
np.random.seed(42)
n_samples = 1000

positive_templates = [
    "excellent product amazing quality",
    "love this great recommend everyone",
    "fantastic service outstanding results",
    "perfect wonderful best purchase"
]
negative_templates = [
    "terrible quality waste money",
    "awful product very disappointed",
    "worst experience do not buy",
    "horrible poor quality avoid"
]

texts = []
labels = []
for i in range(n_samples // 2):
    template = positive_templates[i % len(positive_templates)]
    texts.append(f"{template} {i % 100}")
    labels.append(1)
for i in range(n_samples // 2):
    template = negative_templates[i % len(negative_templates)]
    texts.append(f"{template} {i % 100}")
    labels.append(0)

X_train, X_test, y_train, y_test = train_test_split(
    texts, labels, test_size=0.2, random_state=42
)

# Classical ML Models
ml_models = {
    'TF-IDF + LR': Pipeline([
        ('tfidf', TfidfVectorizer(ngram_range=(1,2))),
        ('clf', LogisticRegression(max_iter=1000))
    ]),
    'TF-IDF + SVM': Pipeline([
        ('tfidf', TfidfVectorizer(ngram_range=(1,2))),
        ('clf', LinearSVC(C=1.0, max_iter=2000))
    ])
}

print("=== Classical ML Performance ===")
for name, model in ml_models.items():
    start = time.time()
    model.fit(X_train, y_train)
    train_time = time.time() - start
    
    start = time.time()
    y_pred = model.predict(X_test)
    infer_time = time.time() - start
    
    f1 = f1_score(y_test, y_pred, average='weighted')
    print(f"{name:20s}: F1={f1:.4f}, Train={train_time:.3f}s, "
          f"Infer={infer_time*1000:.1f}ms")
```

## 5. So sánh Conceptual: Feature Learning

```
Classical ML:
  "I love this movie!" 
  → Tokenize → BoW: {love:1, movie:1} 
  → TF-IDF: {love:0.7, movie:0.3}
  → Human-designed features
  
Deep Learning (BERT):
  "I love this movie!"
  → Tokenize → Token IDs → Embeddings → Self-Attention → Hidden States
  → Tự học context: "love" trong context này = positive
  → "love" trong "love story" ≠ "love this product" (cùng nghĩa hay khác?)
  → BERT hiểu được ngữ cảnh phức tạp hơn
```

## 6. Sarcasm: ML vs DL

```python
# ML thường sai với sarcasm
sarcastic_examples = [
    "Oh great, another Monday!",          # actually negative
    "Wow, such an amazing traffic jam!",   # actually negative
    "Just what I needed, more rain.",      # actually negative
    "This movie was so bad it was good.",  # mixed
]

# TF-IDF + LR: thấy "great", "amazing" → dự đoán Positive (SAI!)
# BERT: hiểu context + tone → dự đoán Negative (ĐÚNG!)

# Testing với model ML
test_model = ml_models['TF-IDF + LR']
preds = test_model.predict(sarcastic_examples)
print("\nML Model trên Sarcasm:")
for text, pred in zip(sarcastic_examples, preds):
    print(f"  Pred: {'Pos' if pred==1 else 'Neg'} | '{text}'")
print("Note: ML thường fail với sarcasm vì không hiểu context!")
```

## 7. Decision Framework

```
Bạn có bài toán NLP:
         ↓
    Dataset < 10k samples?
    ┌──── Yes ────────────────────────────────────→ Classical ML
    │                                               (LR, SVM, NB)
    No
    ↓
    Cần interpretability?
    ┌──── Yes ────────────────────────────────────→ Classical ML
    │                                               + LIME/SHAP
    No
    ↓
    Task phức tạp? (NER, QA, Translation)
    ┌──── Yes ────────────────────────────────────→ Deep Learning
    │                                               (BERT, GPT)
    No
    ↓
    Có GPU + thời gian?
    ┌──── Yes ────────────────────────────────────→ Deep Learning
    │                                               for better perf
    No
    ↓
    Classical ML (nhanh, đủ tốt)
```

## 8. Tóm tắt

**Classical ML vẫn mạnh khi:**
- Small/medium data
- Low latency requirement
- Interpretability needed
- Limited compute

**Deep Learning thắng khi:**
- Large data + compute
- Complex semantic understanding
- Context-aware tasks
- State-of-the-art needed

> **"Always start with Classical ML as baseline, then try DL if needed."**
