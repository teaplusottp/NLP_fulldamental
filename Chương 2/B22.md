## 1. Tại sao Interpretability quan trọng?

**Black box model** = biết output nhưng không biết tại sao.

**Vấn đề trong thực tế:**
- Khách hàng hỏi "tại sao email tôi bị mark là spam?"
- Medical NLP: bác sĩ cần hiểu tại sao hệ thống đưa ra diagnosis
- Legal compliance: GDPR yêu cầu **right to explanation**

## 2. Linear Models: Weights là Explanation

Logistic Regression và LinearSVC có **weights trực tiếp giải thích**:

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.linear_model import LogisticRegression
import numpy as np
import pandas as pd

texts = [
    "win free money prize click here",
    "urgent offer expires limited time",
    "meeting office tomorrow afternoon",
    "project review deadline next week",
    "cheap pills medicine no prescription",
    "hello colleague hope doing well",
    "team lunch cafeteria today 12pm",
    "quarterly report presentation monday"
]
labels = [1,1,0,0,1,0,0,0]  # 1=spam

vec = TfidfVectorizer(ngram_range=(1,2))
X = vec.fit_transform(texts)

model = LogisticRegression(max_iter=1000, random_state=42)
model.fit(X, labels)

feature_names = vec.get_feature_names_out()
coef = model.coef_[0]

# DataFrame để xem rõ
weights_df = pd.DataFrame({
    'feature': feature_names,
    'weight': coef
}).sort_values('weight')

print("Top Spam features:")
print(weights_df.tail(10).to_string())

print("\nTop Ham features:")
print(weights_df.head(10).to_string())
```

## 3. Per-sample Explanation

```python
def explain_prediction(text, model, vectorizer, top_k=5):
    """Giải thích dự đoán cho một sample"""
    vec_transform = vectorizer.transform([text])
    pred = model.predict(vec_transform)[0]
    proba = model.predict_proba(vec_transform)[0]
    
    coef = model.coef_[0]
    feature_names = vectorizer.get_feature_names_out()
    
    # Contribution của từng feature = weight * tfidf_value
    tfidf_values = vec_transform.toarray()[0]
    contributions = coef * tfidf_values
    
    # Sort by absolute contribution
    sorted_idx = np.argsort(np.abs(contributions))[::-1]
    
    print(f"Text: '{text}'")
    print(f"Prediction: {'SPAM' if pred == 1 else 'HAM'} (confidence: {max(proba):.2%})")
    print(f"\nTop {top_k} contributing features:")
    
    for i in sorted_idx[:top_k]:
        if tfidf_values[i] > 0:
            direction = "→ SPAM" if contributions[i] > 0 else "→ HAM"
            print(f"  '{feature_names[i]}': contribution={contributions[i]:.4f} ({direction})")

explain_prediction("win free money click now!", model, vec)
```

## 4. LIME (Local Interpretable Model-Agnostic Explanations)

```python
# pip install lime
from lime.lime_text import LimeTextExplainer
from sklearn.pipeline import Pipeline

pipeline = Pipeline([
    ('tfidf', TfidfVectorizer(ngram_range=(1,2))),
    ('clf', LogisticRegression(max_iter=1000, random_state=42))
])
pipeline.fit(texts, labels)

explainer = LimeTextExplainer(class_names=['Ham', 'Spam'])

# Giải thích cho một instance
test_text = "win money free prizes click here now"
exp = explainer.explain_instance(
    test_text,
    pipeline.predict_proba,
    num_features=6
)

print("LIME Explanation:")
for feature, weight in exp.as_list():
    direction = "→ SPAM +" if weight > 0 else "→ HAM -"
    print(f"  '{feature}': {weight:.4f} ({direction})")

# Visualize (nếu có Jupyter)
# exp.show_in_notebook()
exp.save_to_file('lime_explanation.html')
```

## 5. SHAP cho Text Models

```python
# pip install shap
import shap

# Với linear model, dùng LinearExplainer
vec_simple = TfidfVectorizer(max_features=50)
X_dense = vec_simple.fit_transform(texts).toarray()

model_simple = LogisticRegression(max_iter=1000)
model_simple.fit(X_dense, labels)

explainer_shap = shap.LinearExplainer(model_simple, X_dense)
shap_values = explainer_shap.shap_values(X_dense)

print("SHAP values shape:", shap_values.shape)
print("Sample SHAP values for first text:")

feature_names_simple = vec_simple.get_feature_names_out()
top_shap = np.argsort(np.abs(shap_values[0]))[-5:][::-1]
for i in top_shap:
    if abs(shap_values[0][i]) > 0:
        print(f"  '{feature_names_simple[i]}': SHAP={shap_values[0][i]:.4f}")
```

## 6. Word Cloud theo Class

## 7. Interpretability Spectrum

```
More Interpretable ←──────────────────────────→ Less Interpretable
       │                                                │
  Linear Models      Decision Tree      Random Forest    Neural Networks
  Logistic Reg       (with depth)       XGBoost           BERT, GPT
       │                                                │
  Direct weights    Visualizable       Feature Imp.    Need LIME/SHAP
```

---

## 8. Tóm tắt

- Linear models: **trực tiếp interpret** qua weights
- Per-sample: contribution = weight × feature_value
- **LIME**: local explanation cho bất kỳ model nào
- **SHAP**: Shapley values, tổng quát hơn LIME
- Word cloud: intuitive nhưng không chính xác
- Trong NLP production → LIME/SHAP là chuẩn để explain
