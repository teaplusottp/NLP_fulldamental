## 1. sklearn Pipeline là gì?

**Pipeline** kết hợp nhiều bước preprocessing + model thành **một object duy nhất**.

```
Text → [TF-IDF] → [Feature Selection] → [Model] → Prediction
         ↑              ↑                  ↑
       Step 1         Step 2             Step 3
```

**Lợi ích:**
- Code gọn sạch hơn
- Tránh data leakage (fit chỉ trên train)
- Cross-validation dễ dàng
- Dễ deploy

---

## 2. Data Leakage là gì?

**Vấn đề phổ biến** khi KHÔNG dùng Pipeline:

```python
# ❌ SAI - Data leakage!
vec = TfidfVectorizer()
X_all_vec = vec.fit_transform(all_texts)  # fit trên toàn bộ data!
X_train, X_test = split(X_all_vec)
model.fit(X_train)
model.predict(X_test)  # X_test đã "nhìn thấy" vocab khi fit trên all_texts
```

```python
# ✅ ĐÚNG - Dùng Pipeline
pipeline = Pipeline([('tfidf', TfidfVectorizer()), ('clf', model)])
pipeline.fit(X_train)  # chỉ fit trên train
pipeline.predict(X_test)  # transform test bằng train vocab
```

## 3. Tạo Basic Pipeline

```python
from sklearn.pipeline import Pipeline
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.linear_model import LogisticRegression
from sklearn.svm import LinearSVC
from sklearn.model_selection import train_test_split

texts = [
    "spam win free money prize click",
    "urgent limited offer expires now",
    "meeting at office 3pm tomorrow",
    "project deadline next friday",
    "buy cheap medicine no prescription",
    "you have been selected winner",
    "team lunch cafeteria 12pm today",
    "quarterly report attached please review"
]
labels = [1,1,0,0,1,1,0,0]

X_train, X_test, y_train, y_test = train_test_split(
    texts, labels, test_size=0.25, random_state=42
)

# Pipeline 1: TF-IDF + Logistic Regression
pipe_lr = Pipeline([
    ('tfidf', TfidfVectorizer(ngram_range=(1,2))),
    ('clf', LogisticRegression(max_iter=1000, random_state=42))
])

pipe_lr.fit(X_train, y_train)
print(f"LR Accuracy: {pipe_lr.score(X_test, y_test):.4f}")

# Pipeline 2: TF-IDF + SVM
pipe_svm = Pipeline([
    ('tfidf', TfidfVectorizer(ngram_range=(1,2))),
    ('clf', LinearSVC(C=1.0, random_state=42))
])

pipe_svm.fit(X_train, y_train)
print(f"SVM Accuracy: {pipe_svm.score(X_test, y_test):.4f}")
```

## 4. Pipeline với Feature Selection

```python
from sklearn.feature_selection import SelectKBest, chi2

pipe_full = Pipeline([
    ('tfidf', TfidfVectorizer(ngram_range=(1,2))),
    ('select', SelectKBest(chi2, k=50)),  # top 50 features
    ('clf', LogisticRegression(max_iter=1000, random_state=42))
])

from sklearn.model_selection import cross_val_score
scores = cross_val_score(pipe_full, texts, labels, cv=3, scoring='accuracy')
print(f"Full Pipeline CV: {scores.mean():.4f} ± {scores.std():.4f}")
```

## 5. Access Pipeline Steps

pipe_lr.fit(X_train, y_train)

## 6. Set Parameters cho Pipeline

```python
# Dùng __ để truy cập params của step trong pipeline
pipe_lr.set_params(
    tfidf__ngram_range=(1, 3),
    tfidf__max_features=5000,
    clf__C=0.5
)
pipe_lr.fit(X_train, y_train)
print(f"Updated LR: {pipe_lr.score(X_test, y_test):.4f}")
```

## 7. Pipeline với Custom Transformer

```python
from sklearn.base import BaseEstimator, TransformerMixin
import re

class TextCleaner(BaseEstimator, TransformerMixin):
    """Custom transformer để clean text"""
    
    def fit(self, X, y=None):
        return self  # không cần fit
    
    def transform(self, X):
        cleaned = []
        for text in X:
            text = text.lower()
            text = re.sub(r'[^a-z\s]', '', text)
            text = re.sub(r'\s+', ' ', text).strip()
            cleaned.append(text)
        return cleaned

# Dùng custom transformer trong pipeline
pipe_custom = Pipeline([
    ('cleaner', TextCleaner()),
    ('tfidf', TfidfVectorizer()),
    ('clf', LogisticRegression(max_iter=1000))
])

pipe_custom.fit(X_train, y_train)
print(f"Custom: {pipe_custom.score(X_test, y_test):.4f}")
```

## 8. FeatureUnion: Combine Multiple Features

```python
from sklearn.pipeline import FeatureUnion
from sklearn.decomposition import TruncatedSVD

# Kết hợp unigram + bigram features
feature_union = FeatureUnion([
    ('unigram', TfidfVectorizer(ngram_range=(1,1))),
    ('bigram', TfidfVectorizer(ngram_range=(2,2)))
])

pipe_union = Pipeline([
    ('features', feature_union),
    ('clf', LogisticRegression(max_iter=1000))
])

pipe_union.fit(X_train, y_train)
print(f"FeatureUnion: {pipe_union.score(X_test, y_test):.4f}")
```

## 10. Tóm tắt

- Pipeline = preprocessing + model trong một object
- Tránh data leakage (fit chỉ trên train)
- Dùng `__` để set params cho từng step
- Custom transformers: kế thừa `BaseEstimator + TransformerMixin`
- `FeatureUnion`: kết hợp nhiều feature extractors
- `joblib.dump/load`: save/load pipeline
