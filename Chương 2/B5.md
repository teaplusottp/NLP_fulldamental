## 1. SVM là gì?

**Support Vector Machine (SVM)** tìm **hyperplane** tốt nhất để phân chia các class, với **margin lớn nhất** có thể.

Trong NLP, SVM (đặc biệt với **linear kernel**) là một trong những model mạnh nhất cho text classification.

---

## 2. Ý tưởng trực quan

```
Class A: [●●●]
Class B: [■■■]

SVM tìm đường thẳng (hyperplane) chia A và B
với khoảng cách tối đa tới các điểm gần nhất (support vectors)
```

---

## 3. Support Vectors

**Support Vectors** = các điểm dữ liệu **gần hyperplane nhất**.

Chỉ các support vectors mới ảnh hưởng đến hyperplane → robust với outliers.

---

## 4. Kernel Trick

| Kernel | Dùng khi | Tốc độ |
|--------|----------|--------|
| **Linear** | Text (TF-IDF) | Nhanh nhất |
| RBF | Data phi tuyến | Chậm hơn |
| Polynomial | Ít dùng trong NLP | - |

👉 Với text classification: **dùng Linear SVM** (LinearSVC).

---

## 5. Code Python

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.svm import LinearSVC, SVC
from sklearn.model_selection import train_test_split
from sklearn.metrics import classification_report

texts = [
    "I love this movie, excellent performance",
    "Terrible film, complete waste of money",
    "Amazing story and great acting",
    "Worst movie ever made, boring",
    "Highly entertaining and well directed",
    "Disappointing and poorly written",
    "Brilliant cinematography and plot",
    "Awful experience, do not watch"
]
labels = [1, 0, 1, 0, 1, 0, 1, 0]

X_train, X_test, y_train, y_test = train_test_split(
    texts, labels, test_size=0.25, random_state=42
)

vec = TfidfVectorizer(ngram_range=(1, 2), min_df=1)
X_train_vec = vec.fit_transform(X_train)
X_test_vec = vec.transform(X_test)

# LinearSVC - nhanh hơn SVC với kernel='linear'
model = LinearSVC(C=1.0, max_iter=2000)
model.fit(X_train_vec, y_train)

y_pred = model.predict(X_test_vec)
print(classification_report(y_test, y_pred))
```

---

## 6. Tham số C (Regularization)

- **C lớn**: fit training data chặt hơn → có thể overfit
- **C nhỏ**: margin rộng hơn → regularization mạnh hơn

```python
# Thử nhiều giá trị C
for C in [0.01, 0.1, 1.0, 10.0, 100.0]:
    model = LinearSVC(C=C)
    model.fit(X_train_vec, y_train)
    score = model.score(X_test_vec, y_test)
    print(f"C={C}: accuracy={score:.3f}")
```

---

## 7. Probability Calibration

LinearSVC không cho probability trực tiếp. Dùng `CalibratedClassifierCV`:

```python
from sklearn.calibration import CalibratedClassifierCV

base_model = LinearSVC(C=1.0)
model = CalibratedClassifierCV(base_model)
model.fit(X_train_vec, y_train)

# Bây giờ có predict_proba
proba = model.predict_proba(X_test_vec)
print(proba)  # [P(negative), P(positive)]
```

## 8. So sánh với Logistic Regression

| | SVM | Logistic Regression |
|-|-----|---------------------|
| Tốc độ train | Chậm hơn | Nhanh hơn |
| Hiệu suất text | Thường cao hơn | Gần bằng |
| Probability | Cần calibration | Có sẵn |
| Interpretability | Khó hơn | Dễ hơn |
| Large dataset | Có thể chậm | Scale tốt |

---

## 9. SGDClassifier - SVM scale tốt

Với dataset lớn, dùng `SGDClassifier`:

```python
from sklearn.linear_model import SGDClassifier

# loss='hinge' → SVM; loss='log_loss' → Logistic Regression
model = SGDClassifier(loss='hinge', penalty='l2', alpha=1e-4, max_iter=100)
model.fit(X_train_vec, y_train)
```

---

## 10. Tóm tắt

- SVM với Linear kernel → **state-of-the-art** cho text classification (trước deep learning)
- Tốt với high-dimensional sparse features (TF-IDF)
- C = regularization param quan trọng cần tune
- `LinearSVC` nhanh hơn `SVC(kernel='linear')`
